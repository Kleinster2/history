# Private Equity and Hedge Funds

Private equity and hedge funds are alternative investment vehicles that operate outside traditional markets—and often outside traditional regulation. Hedge funds bet on price movements with leverage and short-selling; private equity buys companies, restructures them, and sells. Both attract the same talent that once went to investment banks, promising outsized returns for sophisticated investors. They are Baumol's thesis in action: the same financial engineering skills can create value through genuine restructuring or extract it through leverage, asset-stripping, and tax arbitrage. Which predominates depends on the institutional environment.

## Overview

### What They Are

| Type | Character |
|------|-----------|
| Hedge funds | Pooled investment, absolute returns, diverse strategies |
| Private equity | Buy companies, restructure, exit |
| Venture capital | Invest in early-stage companies |
| Distinction | Blurring—PE funds have hedge strategies, and vice versa |

### Scale

| Metric | Amount (2023) |
|--------|---------------|
| Hedge fund AUM | ~$4.5 trillion |
| Private equity AUM | ~$8 trillion |
| Venture capital | ~$1 trillion (part of PE) |
| Growth | Both grew 10x since 2000 |

## Hedge Funds

### Origins

| Development | Date/Person |
|-------------|-------------|
| First hedge fund | Alfred Winslow Jones (1949) |
| Innovation | Long-short equity, leverage, incentive fee |
| "Hedged" fund | Name from hedging long with shorts |
| 1960s | Several early funds (Soros, Steinhardt) |

### Key Innovations

| Innovation | Effect |
|------------|--------|
| Long-short | Buy undervalued, short overvalued |
| Leverage | Amplify returns |
| Incentive fee | 2% management + 20% of profits |
| Limited partnership | Legal structure |
| Accredited investors | Avoid SEC registration |

### Major Strategies

| Strategy | Description |
|----------|-------------|
| Long-short equity | Stock picking both directions |
| Global macro | Bet on economies, currencies |
| Event-driven | Mergers, bankruptcies, spin-offs |
| Arbitrage | Price discrepancies (convertible, statistical) |
| Quantitative | Algorithmic, systematic |
| Distressed | Buy troubled debt/equity |
| Activist | Buy stake, push for changes |

## Hedge Fund Economics

### Fee Structure

| Component | Standard |
|-----------|----------|
| Management fee | 2% of AUM annually |
| Performance fee | 20% of profits |
| High-water mark | Only earn on new profits |
| Hurdle rate | Some require beating benchmark |
| Trend | Fees declining (1.5/15 more common now) |

### Returns

| Finding | Evidence |
|---------|----------|
| Average performance | Roughly matches market after fees |
| Dispersion | Top funds massively outperform |
| Persistence | Unclear if past returns predict |
| Risk-adjusted | Mixed—some strategies add alpha |
| Survivorship bias | Failed funds disappear from data |

### Criticisms

| Criticism | Argument |
|-----------|----------|
| Fees too high | 2/20 captures most alpha |
| Returns mediocre | Average is average |
| Crowded trades | Too much money chasing same ideas |
| Systemic risk | Leverage, counterparty exposure |
| Short-termism | Pressure on companies |

## Hedge Fund Crises

### LTCM (1998)

| Element | Description |
|---------|-------------|
| Fund | Long-Term Capital Management |
| Strategy | Arbitrage with massive leverage |
| Partners | Nobel laureates, Salomon veterans |
| Crisis | Russia default, spreads blew out |
| Leverage | 25:1 or higher |
| Bailout | Fed-organized rescue (no public money) |
| Lesson | Leverage + illiquidity = disaster |

### 2008

| Role | Description |
|------|-------------|
| Victims | Many funds crashed |
| Perpetrators | Short-selling financials |
| Redemptions | Investors pulled money |
| Gates | Funds locked up withdrawals |
| Recovery | Most survived, grew larger |

## Private Equity

### Origins

| Development | Date |
|-------------|------|
| Venture capital | 1946 (ARDC) |
| LBO pioneers | 1960s-70s |
| KKR | 1976 (Kohlberg, Kravis, Roberts) |
| 1980s boom | Junk bond financing |
| Mega-deals | RJR Nabisco (1989)—$31B |

### The LBO Model

| Step | Description |
|------|-------------|
| 1. Target | Find undervalued/underperforming company |
| 2. Acquire | Buy using mostly debt (60-80%) |
| 3. Restructure | Cut costs, improve operations |
| 4. Leverage | Debt paid down from cash flow |
| 5. Exit | Sell to strategic buyer, another PE, or IPO |
| Return | Equity gains from debt paydown + value creation |

### Major Firms

| Firm | Founded | Character |
|------|---------|-----------|
| KKR | 1976 | LBO pioneer |
| Blackstone | 1985 | Largest, diversified |
| Carlyle | 1987 | DC connections, defense |
| Apollo | 1990 | Credit, distressed |
| TPG | 1992 | Growth-oriented |
| Bain Capital | 1984 | Consulting heritage |

### Strategies

| Strategy | Description |
|----------|-------------|
| Buyout | Classic LBO—take control |
| Growth equity | Minority stake, growth capital |
| Venture capital | Early-stage, high-risk |
| Distressed | Buy troubled companies/debt |
| Secondaries | Buy existing PE fund stakes |
| Infrastructure | Long-term, stable cash flows |
| Real estate | Property deals |

## The Deeper Point

Private equity and hedge funds are [[Entrepreneurship|Baumol's thesis]] in financial form: the same talent for deal-making, analysis, and capital allocation can create genuine value or extract it through financial engineering, depending on [[Institutions and Progress|institutional]] incentives. When KKR restructures a failing manufacturing company, cuts waste, and returns it to profitability, that's productive entrepreneurship. When a PE firm loads a retailer with debt, extracts dividend recaps, and leaves it unable to invest in e-commerce, that's extraction. The carried interest tax treatment—taxing PE profits as capital gains rather than income—is an institutional choice that channels talent toward financial engineering. The rise of alternative investments reflects institutional gaps: pension funds need returns that public markets don't provide; companies prefer private to public markets where disclosure and short-term pressure are intense. The [[Securities Regulation|regulatory]] framework—light for alternatives, heavy for public markets—may be backwards: the most sophisticated investors in alternatives are the least able to discipline behavior (pension beneficiaries don't choose managers). The 2008 crisis showed hedge funds can create systemic risk; the retail bankruptcy wave shows PE can destroy companies. But VC has genuinely funded [[Innovation Economics|innovation]] that wouldn't have happened otherwise. The industry's effects depend on which activities predominate—and that depends on rules, returns, and the specific funds. It's Baumol all the way down.
