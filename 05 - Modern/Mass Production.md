# Mass Production

Mass production transformed economic life more than any innovation since agriculture. When Henry Ford's assembly line made the Model T affordable, it proved a revolutionary idea: pay workers enough to buy what they make, and the whole economy grows. This "Fordist bargain" defined the 20th centuryâ€”high wages, high output, mass consumption. Then Toyota improved it, China scaled it, and robots began replacing it.

## Overview

### The Transformation

| Measure | Before (1900) | After (1970) |
|---------|---------------|--------------|
| Car production time | 12+ hours | 1 hour |
| Car price (real) | Years of income | Months of income |
| Factory workers | Skilled craftsmen | Semi-skilled operators |
| Output per worker | Low | 10-50x higher |

### Key Innovations

| Innovation | Pioneer | Date |
|------------|---------|------|
| Scientific management | Taylor | 1911 |
| Moving assembly line | Ford | 1913 |
| $5 day (high wages) | Ford | 1914 |
| Just-in-time | Toyota | 1950s |
| Lean manufacturing | Toyota | 1970s |
| Robotic assembly | Global | 1980s+ |

## The Ford System

### The Assembly Line (1913)

| Innovation | Effect |
|------------|--------|
| Moving line | Work comes to worker |
| Standardized parts | Interchangeable |
| Task specialization | Each worker one operation |
| Continuous flow | No stops, no waiting |

### The $5 Day (1914)

| Feature | Significance |
|---------|--------------|
| Doubled wages | From $2.34 to $5.00 |
| Why? | Reduce 370% annual turnover |
| Deeper logic | Workers must afford products |
| Result | Created consumer market |

### Vertical Integration

| Stage | Ford Controlled |
|-------|-----------------|
| Raw materials | Rubber plantations, iron mines |
| Components | Made in-house |
| Assembly | River Rouge complex |
| Distribution | Dealer network |

### Model T Economics

| Measure | 1908 | 1924 |
|---------|------|------|
| Price | $850 | $290 |
| Production | 10,000/year | 2 million/year |
| Market share | Small | 50%+ |

## Toyota Production System

### Origins

| Context | Factor |
|---------|--------|
| Postwar Japan | Small market, no capital |
| Constraint | Couldn't afford mass inventory |
| Solution | Make only what's needed |
| Developer | Taiichi Ohno |

### Key Principles

| Principle | Method |
|-----------|--------|
| Just-in-time | Parts arrive when needed |
| Jidoka | Stop line for defects |
| Kaizen | Continuous improvement |
| Kanban | Visual production signals |
| Muda elimination | Remove all waste |

### How It Differed from Ford

| Dimension | Ford | Toyota |
|-----------|------|--------|
| Inventory | Large buffers | Minimal (JIT) |
| Quality | Inspect at end | Build in at source |
| Workers | Interchangeable | Multi-skilled teams |
| Improvement | Management-driven | Worker-driven (kaizen) |
| Variety | One model | Many models, same line |

### Results

| Measure | Toyota Advantage |
|---------|------------------|
| Defects | 10x fewer |
| Inventory | 90% less |
| Development time | Half |
| Worker suggestions | 100x more |

## Crisis of Fordism (1970s)

### What Went Wrong

| Problem | Manifestation |
|---------|---------------|
| Stagflation | Inflation + unemployment |
| Oil shocks | Energy costs soared |
| Japanese competition | Better quality, lower cost |
| Labor militancy | Strikes, rigidity |
| Market saturation | Everyone had a car |

### Post-Fordism

| Feature | Description |
|---------|-------------|
| Flexible specialization | Small batches, quick change |
| Lean production | Toyota model spreads |
| Global supply chains | Fragment production |
| Just-in-time | Minimize inventory |
| Outsourcing | Focus on core competency |
