# Leveraged Buyouts

How debt became a tool for corporate restructuring—and asset stripping. The LBO boom transformed corporate America, proving that financial engineering could extract value from operating companies. Whether this channeled capital toward efficiency or extraction remains capitalism's central question.

## What Is an LBO?

### The Basic Structure

| Element | Description |
|---------|-------------|
| Target | Operating company |
| Buyer | Financial sponsor (PE firm) |
| Financing | Mostly debt (70-90%) |
| Collateral | Target's own assets |
| Exit | Sell or IPO in 3-7 years |

### The Logic

| Step | Purpose |
|------|---------|
| Borrow heavily | Small equity, large returns |
| Use cash flows | To service debt |
| "Improve" operations | Cut costs, sell assets |
| Deleverage | Pay down debt |
| Exit | Sell at higher multiple |

### Why It Works (When It Does)

| Mechanism | Effect |
|-----------|--------|
| Tax shield | Interest deductible |
| Management discipline | Debt forces efficiency |
| Concentrated ownership | Aligned incentives |
| Exit pressure | Must perform |

## The Deeper Point

Leveraged buyouts exemplify the [[Financialization|financialization]] question: does financial engineering create value or merely redistribute it? The PE industry claims operational improvement justifies the returns. Critics see wealth transfer from workers and communities to financial intermediaries.

The institutional economics lens suggests both can be true. In some cases, concentrated ownership and debt discipline genuinely improve poorly managed companies. In others, tax arbitrage, asset stripping, and cost-cutting extract value without creating it. The question is which effect dominates—and the answer likely varies by deal, sector, and time period.

The deeper issue is what LBOs reveal about corporate governance. If financial sponsors can extract billions through restructuring, either the original managers were incompetent or the restructuring destroyed something valuable (jobs, capabilities, R&D). The first interpretation justifies PE as market discipline. The second sees it as institutional failure—financiers rewarded for short-term extraction rather than long-term building.

[[Institutions and Progress|Institutional]] rules determine which interpretation prevails. Tax rules that favor debt over equity incentivize leverage. Weak labor protections enable cost-cutting. Limited disclosure allows opacity. PE's growth reflects institutional choices that channel capital toward financial engineering rather than patient investment.
