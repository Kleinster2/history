# Corporate Evolution

The corporation is the dominant institution of modern capitalism. From the managerial revolution of the 1920s through conglomerate empires, multinational expansion, shareholder capitalism, and today's platform giants, corporate forms evolved to coordinate ever-larger economic activity. How firms are owned, governed, and organized shapes who benefits from economic progressâ€”shareholders, workers, managers, or society.

## Overview

### The Arc

| Era | Corporate Form | Logic |
|-----|----------------|-------|
| 1870-1920 | Trust/monopoly | Eliminate competition |
| 1920-1960 | Managerial corporation | Professional management |
| 1960-1980 | Conglomerate | Diversify risk |
| 1980-2000 | Shareholder-focused | Maximize stock price |
| 2000-present | Platform/network | Control information |

### Scale of Corporate Power

| Measure | Status |
|---------|--------|
| Largest firms' revenue | Exceed most countries' GDP |
| Walmart employees | 2.1 million |
| Apple market cap | $3+ trillion |
| Fortune 500 revenue | 40% of US GDP |

## The Managerial Revolution (1920s-1960s)

### Berle and Means (1932)

| Observation | Implication |
|-------------|-------------|
| Ownership dispersed | Thousands of shareholders |
| Control separated | Managers run firms |
| Power shift | From owners to managers |
| Accountability gap | Managers serve themselves? |

### The M-Form (Multidivisional)

| Innovation | Description |
|------------|-------------|
| Creator | Alfred Sloan (GM), DuPont |
| Structure | Divisions by product/region |
| Headquarters | Strategy, finance, control |
| Divisions | Operating autonomy |
| Advantage | Manage diversity |

### The Organization Man

| Feature | Character |
|---------|-----------|
| Lifetime employment | Corporate loyalty |
| Internal promotion | Climb the ladder |
| Corporate welfare | Pensions, healthcare |
| Conformity | "Man in the gray flannel suit" |
| Countries | US, UK, Japan (strongest) |

### Managerial Capitalism's Logic

| Stakeholder | Treatment |
|-------------|-----------|
| Shareholders | Adequate dividends |
| Workers | Steady wages, benefits |
| Managers | Career, status, perks |
| Community | Corporate citizen |
| Balance | "Retain and reinvest" |

## The Conglomerate Era (1960s-1980s)

### What Happened

| Feature | Description |
|---------|-------------|
| Diversification | Firms bought unrelated businesses |
| Logic | Portfolio management, financial synergies |
| Champions | ITT, LTV, Gulf+Western, Textron |
| Scale | Conglomerate merger wave |

### Why Conglomerates

| Argument | Logic |
|----------|-------|
| Risk diversification | Spread across sectors |
| Internal capital markets | Allocate better than banks |
| Tax advantages | Offset losses |
| Managerial empire | Growth = power |

### Why They Failed (Mostly)

| Problem | Evidence |
|---------|----------|
| Complexity | Can't manage everything |
| No synergies | Unrelated businesses don't help each other |
| Conglomerate discount | Market valued parts > whole |
| 1980s breakup | Raiders dismantled them |

### Survivors and Variants

| Type | Examples |
|------|----------|
| US (mostly broken up) | GE (eventually), ITT (broken up) |
| Japanese keiretsu | Cross-shareholding groups |
| Korean chaebol | Samsung, Hyundai, LG |
| Indian groups | Tata, Reliance, Birla |
| Developing world | Still common where capital markets weak |

## Multinational Expansion

### Waves of Multinationalization

| Wave | Character |
|------|-----------|
| Pre-1914 | Resource extraction, trading companies |
| 1950s-1970s | US firms expand (manufacturing) |
| 1980s-1990s | European, Japanese firms globalize |
| 2000s+ | Emerging market MNCs (China, India, Brazil) |

### Why Firms Go Multinational

| Motive | Example |
|--------|---------|
| Market access | Sell where customers are |
| Cost reduction | Cheaper labor, resources |
| Knowledge seeking | Access technology |
| Risk diversification | Not all eggs in one basket |
| Regulatory arbitrage | Lower standards |

### How MNCs Operate

| Strategy | Description |
|----------|-------------|
| Export | Sell from home |
| Licensing | Let others produce |
| Joint venture | Partner locally |
| Subsidiary | Own operations abroad |
| Global integration | One world factory |

### Transfer Pricing and Tax

| Issue | Description |
|-------|-------------|
| Internal pricing | Shift profits to low-tax jurisdictions |
| Scale | $500+ billion/year in lost tax revenue |
| Havens | Ireland, Luxembourg, Caymans, Singapore |
| Reform attempts | OECD BEPS, minimum tax |

### MNC Power

| Measure | Scale |
|---------|-------|
| Top 100 firms' revenue | Larger than most countries |
| Employment | Millions across borders |
| Political influence | Lobby, threaten to leave |
| Debate | Do they develop or exploit? |

## The Shareholder Revolution (1980s-2000s)

### The Critique of Managerialism

| Thinker | Argument |
|---------|----------|
| Jensen & Meckling | Managers waste money (agency costs) |
| Friedman | Only obligation is profit |
| Manne | Market for corporate control needed |
| Logic | Shareholders own, managers are agents |

### Hostile Takeovers

| Feature | Description |
|---------|-------------|
| Raiders | Buy undervalued, restructure |
| Champions | Carl Icahn, T. Boone Pickens, KKR |
| LBOs | Debt-financed buyouts |
| Effect | Disciplined managers |
| Cost | Layoffs, communities hurt |

### Shareholder Value Maximization

| Principle | Implementation |
|-----------|----------------|
| Stock price | Primary metric |
| Stock options | Align manager/shareholder interests |
| Buybacks | Return cash to shareholders |
| Downsizing | Cut costs for profit |
| Short-termism | Quarterly earnings focus |

### "Downsize and Distribute"

| Old Model | New Model |
|-----------|-----------|
| Retain and reinvest | Downsize and distribute |
| Workers share gains | Workers expendable |
| Long-term investment | Short-term returns |
| Stakeholder balance | Shareholder primacy |

## Platform Era (2000s-Present)

### What's Different

| Feature | Platform Company |
|---------|------------------|
| Assets | Minimal physical assets |
| Value | Network effects, data |
| Scale | Winner-take-all dynamics |
| Labor | Contractors, gig workers |
| Market power | Natural monopoly tendency |

### The Giants

| Company | Platform Type |
|---------|--------------|
| Apple | Hardware + ecosystem |
| Google/Alphabet | Search + advertising |
| Amazon | Marketplace + cloud |
| Meta | Social network + advertising |
| Microsoft | Enterprise + cloud |

### Chinese Tech Giants

| Company | Domain |
|---------|--------|
| Alibaba | E-commerce |
| Tencent | Social, gaming, payments |
| ByteDance | Short video (TikTok) |
| Huawei | Telecoms, infrastructure |
| State relationship | Complex, regulated |

### Platform Economics

| Feature | Effect |
|---------|--------|

| Data advantages | More data = better service |
| Zero marginal cost | Scale infinitely |
| Multi-sided markets | Connect producers/consumers |
| Lock-in | Switching costs |

## Corporate Governance Compared

### Models

| Model | Countries | Character |
|-------|-----------|-----------|
| Anglo-Saxon | US, UK | Shareholder primacy |
| German | Germany, Austria | Codetermination |
| Japanese | Japan | Stakeholder, cross-shareholding |
| Korean | South Korea | Chaebol, family control |
| Chinese | China | State-private hybrid |
| Nordic | Scandinavia | Stakeholder + strong unions |

### Key Differences

| Dimension | Anglo-Saxon | German | Japanese |
|-----------|-------------|--------|----------|
| Goal | Shareholder value | Stakeholder balance | Long-term growth |
| Board | Independent directors | Worker representatives | Internal promotion |
| Finance | Stock market | Bank relationships | Cross-shareholding |
| Labor | Flexible | Protected, voice | Lifetime (weakening) |
| Time horizon | Quarterly | Long-term | Very long-term |

### Recent Convergence?

| Trend | Direction |
|-------|-----------|
| Shareholder pressure | Spreading globally |
| ESG movement | Pushback on pure shareholder |
| Stakeholder capitalism | Rhetoric returning |
| But | Practice still shareholder-dominated |

## Debates

### Stakeholder vs. Shareholder

| View | Argument |
|------|----------|
| Shareholder | Clear, accountable, efficient |
| Stakeholder | Sustainable, just, stable |
| ESG | Market-based stakeholder approach |
| Reality | Most firms still shareholder-focused |

### Too Big?

| Question | Issue |
|----------|-------|
| Market power | Monopoly/oligopoly concerns |
| Political power | Lobbying, regulatory capture |
| Inequality | CEO pay vs. workers |
| Innovation | Do giants suppress? |
