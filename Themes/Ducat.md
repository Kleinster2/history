# Ducat

The Venetian ducat (*ducato*) was the world's longest-running coin—minted with virtually unchanged specifications for **513 years** (1284-1797). Where the [[Florin]] dominated through banking networks, the ducat dominated through trade routes. Venice's position as the gateway between Europe and the East made the ducat the currency of Mediterranean commerce, accepted from London to Constantinople to India.

## Overview

| Feature | Details |
|---------|---------|
| Name | Ducato (from Latin *ducatus*, duchy) |
| Also called | Zecchino, sequin |
| First minted | October 31, 1284 |
| Last minted | 1797 (fall of Republic) |
| Duration | 513 years |
| Weight | 3.545 grams |
| Purity | 99.47% gold (highest medieval metallurgy could achieve) |
| Diameter | ~21 mm |

## The Coin

### Design

| Side | Image | Inscription |
|------|-------|-------------|
| Obverse | Doge kneeling before St. Mark receiving banner | Doge's name |
| Reverse | Christ in mandorla (almond-shaped glory) | *SIT T XPE DAT QTV REGIS ISTE DVCAT* |

The reverse inscription translates: "To thee, O Christ, be dedicated this duchy which thou rulest."

### Unchanged for 500 Years

| Attribute | Consistency |
|-----------|-------------|
| Weight | Same 3.545g throughout |
| Purity | Same 99.47% throughout |
| Design | Only doge's name changed |
| Size | Same 21mm throughout |

This consistency was deliberate policy. The ducat's value came from predictability.

## Origins

### Why 1284?

| Event | Connection |
|-------|------------|
| 1282 | Sicilian Vespers revolt |
| 1282 | Byzantine Emperor Michael VIII backs revolt |
| 1282 | Michael VIII debases the hyperpyron |
| 1284 | Venice responds with pure gold ducat |

The Byzantine gold coin (hyperpyron) had been the Mediterranean standard. When Byzantium debased it, Venice saw an opportunity.

### Matching the Florin

| Specification | Florin (1252) | Ducat (1284) |
|---------------|---------------|--------------|
| Weight | 3.54g | 3.545g |
| Purity | ~99% | 99.47% |
| Purpose | Interchangeable for trade |

Venice deliberately matched Florentine specifications to compete directly.

## Decline and End

### Why It Lasted So Long

| Factor | Effect |
|--------|--------|
| Republic stability | No civil wars, no regime changes |
| Trade continuity | Venice remained commercial hub |
| State commitment | Never tempted to debase |
| Institutional strength | Mint maintained standards |

### The End (1797)

| Date | Event |
|------|-------|
| May 1797 | Napoleon demands Venice surrender |
| May 12, 1797 | Great Council votes to dissolve Republic |
| October 1797 | Treaty of Campo Formio |
| 1797 | Last true Venetian ducat |

The ducat died with the Republic—same as the florin and Florence.

The ducat's 513-year run is unmatched in monetary history.

## The Deeper Point

The ducat outlasted the florin because Venice outlasted Florence. Both coins maintained purity; both were backed by sophisticated commercial systems; both were trusted internationally. The difference was political: Florence's Republic fell in 1532, ending the florin; Venice's Republic survived until 1797, preserving the ducat.

This suggests that currency stability ultimately depends on institutional stability. The ducat wasn't better gold—it was the same gold. But Venice's institutions (the Great Council, the Doge system, the trading companies) proved more durable than Florence's. When those institutions finally fell to Napoleon, the ducat fell with them.

The ducat also shows the power of path dependence. Once established as the Mediterranean trade currency, the ducat's position reinforced itself. Merchants held ducats because other merchants accepted ducats. This network effect sustained Venetian monetary hegemony even as Venice's relative economic power declined in the 16th-17th centuries.
