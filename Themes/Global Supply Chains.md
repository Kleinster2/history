# Global Supply Chains

Global supply chains transformed how the world makes things. Instead of producing goods in one place, modern manufacturing fragments production across continentsâ€”design in California, components from Taiwan and Korea, assembly in China, shipping through Singapore, sale everywhere. This "unbundling" created efficiency and vulnerability, prosperity and dependence, winners and losers.

## Overview

### What Supply Chains Are

| Term | Meaning |
|------|---------|
| Supply chain | All steps from raw material to consumer |
| Global | Spanning multiple countries |
| Fragmentation | Breaking production into stages |
| Coordination | Managing the pieces |

### The Scale

| Measure | Status |
|---------|--------|
| Trade in intermediates | 2/3 of world trade |
| Countries in iPhone | 43 |
| Container ships | 50,000+ at sea |
| Trade/GDP ratio | 50%+ for many countries |

## Historical Background

### Before Globalization

| Era | Production |
|-----|------------|
| Craft | One place, one artisan |
| Factory | One place, many workers |
| National | Components and assembly together |

### First Unbundling (1820-1990)

| Pattern | Description |
|---------|-------------|
| Production | Concentrated in factories |
| Trade | Final goods |
| Innovation | Steam, then internal combustion |
| Location | Near markets |

### Second Unbundling (1990-Present)

| Pattern | Description |
|---------|-------------|
| Production | Fragmented globally |
| Trade | Components, intermediates |
| Innovation | ICT, containerization |
| Location | Cheapest effective place |

## How Supply Chains Work

### Typical Flow

| Stage | Example (iPhone) |
|-------|------------------|
| Design | California |
| Chips | Taiwan (TSMC) |
| Display | Korea (Samsung) |
| Memory | Japan, Korea |
| Assembly | China (Foxconn) |
| Distribution | Global |

### Coordination Mechanisms

| Mechanism | Description |
|-----------|-------------|
| Lead firms | Apple, Nike, Walmart |
| Suppliers | Tiered (1st, 2nd, 3rd) |
| Contracts | Specifications, timing |
| Standards | Quality, compliance |

### Just-in-Time

| Feature | Description |
|---------|-------------|
| Minimal inventory | Arrives when needed |
| Efficiency | Reduces waste |
| Fragility | No buffer |
| Toyota origin | Spread globally |

## Winners and Losers

### Winners

| Beneficiary | Gain |
|-------------|------|
| Consumers | Lower prices |
| Emerging markets | Jobs, technology |
| Lead firms | Higher profits |
| Shareholders | Value captured |
| Logistics firms | New industry |

### Losers

| Loser | Loss |
|-------|------|

| Some regions | Deindustrialized |

| Small producers | Squeezed by buyers |

| Effect | Description |

## Vulnerabilities

### Revealed by Shocks

| Shock | Lesson |
|-------|--------|
| 2011 Japan earthquake | Single-source risk |
| 2011 Thailand floods | Hard drive shortage |
| COVID-19 (2020) | Everything disrupted |
| Suez blockage (2021) | Chokepoint risk |
| Ukraine war (2022) | Energy, food supply |

### Types of Risk

| Risk | Example |
|------|---------|
| Concentration | Taiwan makes 90% of advanced chips |
| Geopolitical | US-China tensions |
| Natural disaster | Earthquake, pandemic |
| Chokepoints | Malacca Strait, Suez |
| Complexity | Can't trace suppliers |

## Reshoring and Resilience

### Post-COVID Shifts

| Trend | Description |
|-------|-------------|
| Reshoring | Bringing production home |
| Near-shoring | Moving closer |
| Friend-shoring | Allied countries |
| Diversification | Multiple sources |
| Just-in-case | More inventory |

### Drivers of Change

| Factor | Effect |
|--------|--------|
| COVID disruption | Exposed vulnerabilities |
| US-China tensions | Decoupling pressure |
| Automation | Makes reshoring viable |
| ESG concerns | Scrutiny of supply chains |
| National security | Chips, pharma |

### Limits of Reshoring

| Constraint | Description |
|------------|-------------|
| Cost | Still cheaper abroad |
| Ecosystems | Suppliers aren't home |
| Skills | Workers retired |
| Scale | Can't replicate Asia |
| Time | Takes years |
