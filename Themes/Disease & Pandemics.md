# Disease & Pandemics

Disease has shaped history as decisively as any war or revolution. The Antonine Plague weakened Rome, the Black Death transformed medieval Europe, and smallpox conquered the Americas for Spain. Epidemics kill millions, collapse economies, and reshape societies. Understanding disease history reveals the vulnerability of all civilizations and the intimate connection between biology and economy.

## Major Pandemics

### Antonine Plague (165-180 CE)

| Feature | Description |
|---------|-------------|
| Disease | Smallpox or measles |
| Deaths | 5-10 million |
| Location | Roman Empire |
| Effect | Military, economic decline |

### Plague of Justinian (541-542 CE)

| Feature | Description |
|---------|-------------|
| Disease | Bubonic plague |
| Deaths | 25-50 million |
| Location | Byzantine, Mediterranean |
| Effect | Ended reconquest hopes |

### Black Death (1347-1351)

| Feature | Description |
|---------|-------------|
| Disease | Bubonic plague |
| Deaths | 75-200 million (30-60% of Europe) |
| Location | Europe, Asia, Africa |
| Effect | Transformed medieval society |

|---------|-------------|
| Diseases | Smallpox, measles, typhus |

| Feature | Description |
|---------|-------------|
