# Agent Instructions

This repository is the history Obsidian vault. Codex and OpenClaw-backed Codex sessions working in this directory must read and follow `CLAUDE.md`; it is the source of truth for vault structure, note style, changelog discipline, link hygiene, human-note placement, and thinker-note quality.

Key rules to preserve:
- After creating or updating notes, log the change in `Changelog.md`.
- Do not create new notes in `People/`; use `Figures/` for historical actors and `Thinkers/` for interpretive or framework-building people.
- A `Thinkers/` note is not done if it is only a source card or graph pointer. Include core thesis, mechanisms, cases/evidence, implications, critiques/limits, and vault use. For named frameworks, state both what the framework predicts and what mechanisms can change the outcome.
- Protect local user changes. Do not revert unrelated modified files.
