# Greek Economy

The ancient Greeks created the Mediterranean's most commercialized economyâ€”and the first systematic thinking about economic life. Yet this was also an economy built on slavery, suspicious of profit-seeking, and fragmented among hundreds of independent city-states. Understanding Greek economics means grappling with these contradictions.

## The Greek Paradox

| Achievement | Limitation |
|-------------|------------|
| Coined money spread | No unified currency |
| Active markets | Citizens avoided trade |
| Banking emerged | Interest morally suspect |
| Economic theory | Condemned commerce |
| Commercial sophistication | Based on slave labor |

## Slavery: The Economic Foundation

### Scale

| Estimate | Description |
|----------|-------------|
| Athens | 80,000-100,000 slaves (~30% population) |
| Sparta | Helots outnumbered citizens |
| Chios | Allegedly first slave market |

### Sources of Slaves

| Source | Method |
|--------|--------|
| War captives | Conquest |
| Piracy | Kidnapping |
| Birth | Children of slaves |
| Debt (non-Greek) | Foreign practice |
| Trade | Purchased |

### Types of Slavery

| Type | Context |
|------|---------|
| Household | Domestic service |
| Agricultural | Farm labor |
| Mining | Laurion silver mines |
| Craft | Workshop production |
| Public | State-owned (demosioi) |

### Economic Role

| Function | Description |
|----------|-------------|
| Production | Agriculture, manufacture |
| Services | Household, banking |
| State revenue | Mining leases |
| Citizen leisure | Enabled politics |

### Conditions

| Context | Treatment |
|---------|-----------|
| Household | Variable, sometimes good |
| Crafts | Skilled, some autonomy |
| Mining | Brutal, short lives |
| Helots | Agricultural serfs |
