# Dholavira

Dholavira was one of the five largest cities of the Indus Valley Civilization and its most impressive example of water management engineering. Located on an island in the Rann of Kutch (Gujarat), this city survived in an arid environment through sophisticated reservoir systemsâ€”an extraordinary investment in infrastructure.

## Water Management

### The Challenge

| Factor | Problem |
|--------|---------|
| Location | Semi-arid island |
| Rainfall | Seasonal, unreliable |
| Population | Thousands to support |
| Solution | Catch and store every drop |

### Reservoir System

| Feature | Description |
|---------|-------------|
| Number | 16 reservoirs identified |
| Location | Along city walls, channels |
| Capacity | Massive water storage |
| Engineering | Rock-cut and built |

### Water Sources

| Source | Method |
|--------|--------|
| Monsoon runoff | Channeled into reservoirs |
| Seasonal streams | Diverted |
| Storm drains | Collected in tanks |

### Reservoir Construction

| Feature | Description |
|---------|-------------|
| Rock-cut | Carved from bedrock |
| Stone-lined | Masonry walls |
| Steps | Access for drawing water |
| Sluices | Water level control |

### Economic Implications

| Implication | Description |
|-------------|-------------|
| Labor investment | Enormous construction effort |
| Long-term planning | Multi-generational project |
| Collective action | Required coordination |
| Settlement viability | Made city possible |
