# Shang Dynasty

The Shang dynasty represents China's first historically verifiable dynasty and the height of Chinese Bronze Age culture. Famous for oracle bone divination and stunning ritual bronzes, the Shang created a distinctive elite economy centered on royal ritual rather than commerce.

## Overview

| Aspect | Details |
|--------|---------|
| Period | c. 1600-1046 BCE |
| Capital(s) | Multiple; Anyang (late) |
| Territory | Yellow River valley |
| Population | Unknown; modest |
| Script | Oracle bone script |
| End | Conquered by Zhou |

## Legacy for Chinese Economy

### What Shang Established

| Pattern | Later Influence |
|---------|-----------------|
| State craft workshops | Continued tradition |
| Ritual legitimacy | Bronze = authority |
| Centralized control | Model for later states |
| Writing | Continuous tradition |

### What Changed

| Shang | Zhou/Later |
|-------|------------|
| Bronze ritual focus | Eventually iron tools |
| Sacrifice economy | Moderated |
| Captive labor | Less central |
| Gift redistribution | Markets emerge |
