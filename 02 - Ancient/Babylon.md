# Babylon

Babylon was one of the most famous cities of the ancient world, serving as the political, cultural, and religious center of Mesopotamia for over a millennium. From Hammurabi's lawcode to Nebuchadnezzar's legendary gardens, Babylon became synonymous with power, wealth, and civilization itself.

## Overview

| Aspect | Details |
|--------|---------|
| Location | Central Mesopotamia, on Euphrates |
| Modern site | Hillah, Iraq |
| Founded | c. 2300 BCE (minor town) |
| Rise | 1894 BCE (Amorite dynasty) |
| Peak periods | Old Babylonian, Neo-Babylonian |
| End | 539 BCE (Persian conquest) |
| Patron deity | Marduk |

### Old Babylonian Economy

| Feature | Description |
|---------|-------------|
| Private enterprise | Merchants, craftsmen dominant |
| Banking | Temples and private families |
| Contracts | Extensive legal documentation |
| Interest rates | Silver: 20%; Grain: 33.3% |
| Property | Buying, selling, renting recorded |

| Topic | Regulation |
|-------|------------|
| Wages | Fixed rates for laborers |
| Prices | Some price controls |
| Interest | Maximum rates set |
| Debt | Release provisions |
| Property | Inheritance, sale rules |
| Commerce | Merchant responsibilities |
