# Iron Age

The Iron Age began when the Bronze Age endedâ€”when iron replaced bronze as the primary metal for tools and weapons. Iron ore is more abundant than copper and tin, making metal technology accessible to more societies. The Iron Age saw the rise of new empires (Assyria, Persia), new alphabets, new religions, and wider use of coinage. Iron democratized metal.

## Overview

| Feature | Description |
|---------|-------------|
| Definition | Iron as primary working metal |
| Dates | ~1200-500 BCE (varies by region) |
| Transition | From Bronze Age collapse |
| Significance | Widespread metal access |

## Iron vs. Bronze

### Advantages of Iron

| Advantage | Description |
|-----------|-------------|
| Abundance | Iron ore more common |
| No tin trade | Self-sufficient |
| Harder (when steeled) | Superior tools |
| Cheaper | Eventually |

### Disadvantages

| Disadvantage | Description |
|--------------|-------------|
| Higher melting point | Harder to work |
| Rust | Corrodes |
| Early iron | Not as hard as bronze |
| Technology | Needed development |

## Technology

### Iron Working

| Stage | Process |
|-------|---------|
| Smelting | Bloom (spongy mass) |
| Working | Hammering, reheating |
| Carburizing | Adding carbon (steel) |
| Quenching | Hardening |

### Steel Development

| Advance | Effect |
|---------|--------|
| Carburization | Harder metal |
| Quenching | Stronger edge |
| Control | Quality improvement |
| Full steel | Later development |
