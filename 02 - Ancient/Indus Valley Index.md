# Indus Valley Index

The Indus Valley Civilization (also called Harappan Civilization) was one of the world's earliest urban societies, flourishing in the northwestern regions of South Asia from roughly 3300 to 1300 BCE. At its peak, it was the largest of the ancient civilizations, with over 1,500 known settlements spanning from Afghanistan to Gujarat.

## The Economic Mystery

The Indus Valley presents a profound puzzle for economic history:

| Feature | What We'd Expect | What We Find |
|---------|------------------|--------------|
| Rulers | Palaces, royal tombs | None found |
| Religion | Temples, priests | No clear temples |
| Administration | Written records | Undeciphered script |
| Inequality | Rich/poor housing | Remarkably uniform |
| Trade | Local exchange | Standardized across 1000+ km |

**How did they coordinate a civilization of 5 million people without visible rulers, temples, or readable records?**

## Economy

> See [[Indus Valley Economy]] for full treatment.

### The Standardization Mystery

| Standard | Scale |
|----------|-------|
| Brick ratio | 1:2:4 across ALL sites |
| Weights | Same system everywhere |
| City planning | Grid pattern repeated |
| Script | Same symbols everywhere |

This implies coordination without visible hierarchy.

### Key Economic Features

| Feature | Description |
|---------|-------------|
| Standardized weights | Cubical, binary/decimal system |
| Seals | 3,500+ found, trade markers |
| No coinage | Weight-based exchange |
| Craft specialization | Beads, metalwork, pottery |
| Long-distance trade | Mesopotamia, Central Asia, Gujarat |

### Trade Networks

| Partner | Evidence |
|---------|----------|
| Mesopotamia | Indus seals found at Ur |
| Oman/Magan | Copper trade |
| Afghanistan | Lapis lazuli |
| Central Asia | Various goods |
| Internal | Standardized across region |

## Decline and Collapse

### Theories

| Theory | Evidence |
|--------|----------|
| Climate change | River shifts, drought |
| River diversion | Ghaggar-Hakra dried up |
| Flooding | Some sites show flood damage |
| Aryan invasion | Largely discredited |
| Economic collapse | Trade disruption |
| Gradual transformation | Not sudden end |

### What Happened

| Phase | Development |
|-------|-------------|
| 1900 BCE | Cities shrink |
| 1800-1700 BCE | Standardization breaks down |
| 1500-1300 BCE | Rural, de-urbanized |
| After | Cemetery H culture, regional |
