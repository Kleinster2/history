# Late Antiquity

Late Antiquity marks the transition from the [[Classical]] Mediterranean world to the medieval periodâ€”roughly 300 to 500 CE in the West, longer in the East. The Roman economic system contracted: trade networks shrank, cities declined, currency debased, and the centralized state fragmented. But new institutions emerged, especially the Church, which would carry economic functions into the medieval world.

## Economic Contraction

### Scale of Decline

| Indicator | Change |
|-----------|--------|
| Mediterranean trade | Declined 80%+ |
| Urban population | Dramatic fall |
| Coinage circulation | Reduced, debased |
| Long-distance trade | Collapsed in West |

### Regional Variation

| Region | Outcome |
|--------|---------|
| Western Empire | Fragmented, ruralized |
| Eastern Empire | Continued (Byzantium) |
| Persia | Sasanian prosperity |
| Frontiers | New kingdoms |

## Causes of Decline

### Internal Factors

| Factor | Effect |
|--------|--------|
| Currency debasement | Inflation, barter |
| Tax burden | Flight from cities |
| Civil wars | Disruption, insecurity |
| Bureaucratic cost | Unsustainable extraction |

### External Factors

| Factor | Effect |
|--------|--------|
| Migrations | Disruption, new rulers |
| Frontier pressure | Military spending |
| Trade disruption | Piracy, warfare |
| Plague of Justinian | Population loss (541 CE) |

## Structural Transformation

### Ruralization

| Change | Description |
|--------|-------------|
| City decline | Population, services reduced |
| Villa economy | Self-sufficient estates |
| Coloni | Tied peasants (proto-serfdom) |
| Local production | Trade networks abandoned |

### Currency Collapse

| Period | Development |
|--------|-------------|
| 3rd century | Debasement crisis |
| Diocletian | Reform attempt, price controls |
| 4th-5th century | Gold for taxes, barter locally |
| Post-Roman West | Near-total demonetization |

### Labor Transition

| From | To |
|------|-----|
| Slavery | Declining (supply dried up) |
| Free labor | Tied to land (coloni) |
| Urban crafts | Rural, estate-based |
| Market labor | Obligation, tribute |

## New Institutions

### The Church

| Function | Economic Role |
|----------|---------------|
| Landowner | Largest in medieval Europe |
| Welfare | Poor relief, hospitals |
| Education | Preserved literacy |
| Banking | Early credit functions |

### Monasteries

| Function | Economic Role |
|----------|---------------|
| Agriculture | Preserved techniques |
| Production | Workshops, copying |
| Land management | Systematic cultivation |
| Network | Trans-regional connections |

### Barbarian Kingdoms

| Kingdom | Economic Character |
|---------|-------------------|
| Visigoths | Roman continuity |
| Ostrogoths | Maintained trade briefly |
| Franks | Ruralized quickly |
| Vandals | Disrupted Mediterranean |

## What Survived

### Continuities

| Element | Survival |
|---------|----------|
| Roman law | Adapted, preserved |
| Church structure | Expanded |
| Eastern Empire | Continued as Byzantium |
| Mediterranean trade | Reduced but not dead |

### What Was Lost

| Element | Outcome |
|---------|---------|
| Unified currency | Fragmented |
| Long-distance trade | Collapsed in West |
| Urban culture | Drastically reduced |
| Literacy | Confined to Church |

## Eastern Survival

### Byzantine Economy

| Feature | Description |
|---------|-------------|
| Constantinople | Continued as trade hub |
| Gold solidus | Stable currency |
| State control | Guilds, regulation |
| Trade | Eastern Mediterranean, Silk Road |

### Sasanian Persia

| Feature | Description |
|---------|-------------|
| Silk Road | Eastern terminus |
| Agriculture | Irrigation systems |
| Trade | Indian Ocean connections |
| Coinage | Silver economy |

## Economic Significance

| Significance | Description |
|--------------|-------------|
| System collapse | Shows fragility of complex economies |
| Institutional transfer | Church carried functions forward |
| Path dependency | Shaped medieval economy |
| Regional divergence | East/West split begins |
