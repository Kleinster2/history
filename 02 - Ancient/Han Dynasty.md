# Han Dynasty

The Han dynasty (206 BCE - 220 CE) created China's first long-lasting unified empire and established economic institutions that shaped Chinese history for two millennia. From state monopolies to the Silk Road, from the "ever-normal granary" to the Salt and Iron Debate, the Han set patterns of economic governance still studied today.

## Overview

| Aspect | Details |
|--------|---------|
| Period | 206 BCE - 220 CE |
| Capital | Chang'an (Western), Luoyang (Eastern) |
| Population | ~57 million (2 CE census) |
| Territory | China proper plus expansions |
| Division | Western Han (206 BCE-9 CE), Eastern Han (25-220 CE) |

## The Salt and Iron Debate (81 BCE)

### Context

| Issue | Background |
|-------|------------|
| Date | 81 BCE |
| Occasion | Court conference |
| Topic | Continue monopolies? |
| Document | *Yantie Lun* (Discourses on Salt and Iron) |

### Positions

| For (Officials) | Against (Confucians) |
|-----------------|---------------------|
| State needs revenue | Burdens the people |
| Defense requires resources | Quality has declined |
| Prevents profiteering | Creates corruption |
| Controls strategic goods | Scholars shouldn't discuss profit |

### Arguments in Detail

**For Monopolies:**
| Argument | Claim |
|----------|-------|
| Military necessity | Xiongnu threat requires revenue |
| Market failure | Merchants exploit shortages |
| State efficiency | Coordination possible |
| Historical precedent | Guan Zhong's policies |

**Against Monopolies:**
| Argument | Claim |
|----------|-------|
| Quality decline | State products inferior |
| Corruption | Officials profit |
| Price gouging | State charges more |
| Moral principle | Profit corrupts |

### Outcome

| Decision | Long-term |
|----------|-----------|
| Monopolies continued | Some relaxation |
| Alcohol ended | Too unpopular |
| Text preserved | Classic of economic debate |

## Granary System

### "Ever-Normal" Granary (Changpingcang)

| Purpose | Mechanism |
|---------|-----------|
| Price stabilization | Buy in surplus, sell in shortage |
| Famine relief | Emergency stores |
| Market intervention | Prevent speculation |

### Operation

| Phase | Action |
|-------|--------|
| Harvest (low prices) | Government buys grain |
| Shortage (high prices) | Government sells grain |
| Result | Prices smoothed |

### Problems

| Issue | Description |
|-------|-------------|
| Corruption | Officials profit |
| Storage costs | Grain spoils |
| Timing | Hard to predict |
| Local needs | Central control difficult |
