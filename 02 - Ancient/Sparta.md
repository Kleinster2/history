# Sparta

Sparta represents the anti-commercial extreme of Greek economic organization. Where Athens embraced trade, coinage, and foreign residents, Sparta rejected them allâ€”creating a militarized autarky based on serf labor (helots) and an ideology of austerity. Sparta shows that ancient states could choose radically different economic paths.

## Overview

| Aspect | Details |
|--------|---------|
| Location | Laconia, Peloponnese |
| Peak population | ~8,000 citizens, 200,000+ helots |
| Citizens | Spartiates (full citizens) |
| Government | Mixed (kings, council, assembly) |
| Economy | Agricultural autarky |
| Character | Military state |

---

## The Economic Choice

### Sparta vs. Athens

| Feature | Sparta | Athens |
|---------|--------|--------|
| Coinage | Rejected (allegedly iron) | Silver owls |
| Trade | Minimal | Extensive |
| Foreigners | Excluded | Metics welcomed |
| Luxury | Banned | Enjoyed |
| Markets | Limited | Agora central |
| Labor | Helots | Slaves, citizens, metics |
| Ideal | Autarky | Commerce |

### Why This Matters

| Lesson | Implication |
|--------|-------------|
| Choice existed | Economies aren't inevitable |
| Trade-offs | Military power vs. wealth |
| Ideology shapes economy | Values determine system |
| Sustainability? | Sparta eventually failed |

## The Helot System

### What Helots Were

| Feature | Description |
|---------|-------------|
| Status | State-owned serfs |
| Origin | Conquered Messenians, Laconians |
| Number | 7-10 helots per Spartiate |
| Location | Worked the land |
| Obligation | Delivered fixed portion of harvest |

### Economic Function

| Role | Description |
|------|-------------|
| Agricultural labor | All farming |
| Surplus extraction | Fed Spartiate class |
| Freed citizens | For military training |
| State ownership | Not private slaves |

### The Helot Economy

| Feature | Description |
|---------|-------------|
| Land allocation | Each Spartiate had a kleros (lot) |
| Fixed rent | Set portion to master |
| Surplus kept | Helots kept remainder |
| Incentive | Some wealth accumulation possible |

### Control Mechanisms

| Method | Purpose |
|--------|---------|
| Krypteia | Secret police/terror |
| Annual war declaration | Legal to kill helots |
| Outnumbered | Spartiates vastly outnumbered |
| Constant vigilance | Military society required |

## Spartiate Economy

### The Land System

| Feature | Description |
|---------|-------------|
| Kleros | Land allotment per citizen |
| Inalienable | Couldn't sell (in theory) |
| Helot-worked | No citizen labor |
| Equality ideal | Same-sized lots (theory) |

### What Spartiates Did

| Activity | Description |
|----------|-------------|
| Military training | Full-time from age 7 |
| Warfare | Primary occupation |
| Mess contribution | Required for citizenship |
| Politics | Assembly participation |

### The Syssitia (Mess System)

| Feature | Description |
|---------|-------------|
| Requirement | Monthly food contribution |
| Failure | Loss of citizenship |
| Function | Equality enforcement |
| Products | Barley, wine, cheese, figs |

### Citizenship Economics

| Requirement | Economic Implication |
|-------------|---------------------|
| Land ownership | Must have kleros |
| Mess contribution | Must produce surplus |
| Full-time military | No other occupation |
| Failure | Demotion to "inferiors" |

## The Iron Money Legend

### The Tradition

| Claim | Source |
|-------|--------|
| Iron bars as currency | Ancient sources (Plutarch) |
| Deliberately heavy | Discourage accumulation |
| Quenched in vinegar | Made useless for tools |
| No gold/silver | Foreign money banned |

### Modern Assessment

| Reality | Evidence |
|---------|----------|
| Probably exaggerated | Some silver found |
| Ideological claim | Austere image |
| Partial truth | Limited monetary economy |
| External trade | Used real money abroad |

### Economic Effect

| Consequence | Description |
|-------------|-------------|
| Limited commerce | Markets restricted |
| No banking | No monetary accumulation |
| Autarky encouraged | Self-sufficiency |
| Foreign trade difficult | By design |

## Austerity Ideology

### Sumptuary Laws

| Restriction | Purpose |
|-------------|---------|
| Simple housing | No luxury display |
| Plain clothing | Equality |
| Simple food | Mess system |
| No gold/silver ornaments | Wealth denial |

### The Spartan Ideal

| Value | Economic Implication |
|-------|---------------------|
| Equality | No visible wealth differences |
| Simplicity | Low consumption |
| Military focus | Non-productive investment |
| Autarky | Self-sufficiency |

### Reality vs. Ideal

| Ideal | Reality |
|-------|---------|
| Equal lots | Inequality grew over time |
| No wealth | Wealth accumulated (hidden) |
| Austere living | Kings and rich had more |
| No trade | Perioikoi traded |

## Military Economy

### The Economic Logic

| Investment | Return |
|------------|--------|
| Full-time soldiers | Military superiority |
| Helot labor | Freed citizens for war |
| Austerity | Resources for military |
| Autarky | No supply dependencies |

### Costs

| Cost | Description |
|------|-------------|
| No economic growth | Static system |
| Population decline | Citizenship losses |
| Innovation suppressed | Change resisted |
| Brittleness | System couldn't adapt |

### Military as Economy

| Feature | Description |
|---------|-------------|
| Main "export" | Mercenary commanders |
| Tribute | From allies/subordinates |
| Plunder | War profits |
| Prestige | Commanded respect/payments |

## Decline

### Economic Factors

| Problem | Effect |
|---------|--------|
| Land concentration | Fewer citizens qualified |
| Population decline | Spartiate numbers fell |
| Wealth inequality | Despite ideology |
| Inflexibility | Couldn't adapt |

### The Numbers

| Date | Spartiates |
|------|------------|
| 480 BCE | ~8,000 |
| 371 BCE | ~1,500 |
| 244 BCE | ~700 |

### Why Decline?

| Factor | Economic Aspect |
|--------|-----------------|
| Land sold/concentrated | Despite prohibition |
| Mess contributions | Many couldn't afford |
| War losses | Not replaced |
| System rigidity | No reform mechanism |

### Reform Attempts

| Reformer | Attempt |
|----------|---------|
| Agis IV (244 BCE) | Land redistribution |
| Cleomenes III (227 BCE) | Debt cancellation |
| Result | Both failed, killed |

## Economic Legacy

### What Sparta Shows

| Lesson | Description |
|--------|-------------|
| Alternatives exist | Not one economic path |
| Trade-offs real | Military vs. prosperity |
| Ideology can dominate | Over economic "rationality" |
| Equality difficult | Even when idealized |

### Why It Failed

| Weakness | Consequence |
|----------|-------------|
| No growth mechanism | Static population needed |
| No flexibility | Couldn't reform |
| Labor coercion | Required constant force |
| Demographic trap | Citizen numbers collapsed |
