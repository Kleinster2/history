# Epic of Gilgamesh

The Epic of Gilgamesh is humanity's oldest surviving major literary workâ€”a Mesopotamian poem about a king who seeks immortality and finds wisdom instead. Written on clay tablets over millennia, it includes a flood story that predates and parallels the biblical account. The epic reveals much about ancient Mesopotamian society, religion, and the human confrontation with mortality.

## Overview

| Feature | Description |
|---------|-------------|
| Genre | Epic poetry |

## The Story

### Tablet I-II: Gilgamesh and Enkidu

| Event | Description |
|-------|-------------|
| Gilgamesh | King of Uruk, 2/3 god |
| Tyranny | Oppresses his people |
| Enkidu created | Wild man, Gilgamesh's equal |
| Civilized | Through a woman |
| Contest | They fight, become friends |

### Tablet III-V: Cedar Forest

| Event | Description |
|-------|-------------|
| Quest | Journey to Cedar Forest |
| Humbaba | Guardian monster |
| Victory | Kill Humbaba |
| Cedars | Cut for Uruk |

### Tablet VI: Ishtar's Proposal

| Event | Description |
|-------|-------------|
| Ishtar | Goddess offers marriage |
| Rejection | Gilgamesh refuses |
| Bull of Heaven | Sent to punish |
| Victory | Kill the bull |

### Tablet VII-VIII: Enkidu's Death

| Event | Description |
|-------|-------------|
| Gods' decree | Enkidu must die |
| Illness | Twelve days |
| Death | Gilgamesh devastated |
| Mourning | Extended grief |

### Tablet IX-X: Quest for Immortality

| Event | Description |
|-------|-------------|
| Fear of death | Gilgamesh terrified |
| Journey | To edge of world |
| Utnapishtim | Survivor of Flood |
| Advice | Accept mortality |

### Tablet XI: The Flood Story

| Event | Description |
|-------|-------------|
| Warning | Gods plan flood |
| Boat | Built to survive |
| Flood | Seven days |
| Survival | Utnapishtim, family, animals |
| Immortality | Granted to Utnapishtim alone |

### Tablet XII: Return

| Event | Description |
|-------|-------------|
| Plant of youth | Found, then lost |
| Return to Uruk | Acceptance |
| Walls | His immortality |
| Wisdom | What he learned |
