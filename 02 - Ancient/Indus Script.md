# Indus Script

The Indus script remains one of the great undeciphered writing systems of the ancient world. Found primarily on seals used in trade, this script is central to the economic mystery of the Indus Valley Civilizationâ€”a sophisticated trading society whose administrative records we cannot read.

## Overview

| Aspect | Details |
|--------|---------|
| Period | c. 2600-1900 BCE |
| Corpus | ~4,000+ inscribed objects |
| Signs | ~400-600 distinct signs |
| Average length | ~5 signs |
| Longest | ~26 signs (rare) |
| Status | Undeciphered |
| Language | Unknown |

---

## The Decipherment Problem

### Why It Remains Undeciphered

| Challenge | Description |
|-----------|-------------|
| No bilingual | No Rosetta Stone equivalent |
| Short texts | Average 5 signs = little context |
| Unknown language | Can't work backward from known |
| No surviving literature | Only brief inscriptions |
| Isolation | No related scripts nearby |

### Compared to Successful Decipherments

| Script | Key to Decipherment |
|--------|---------------------|
| Egyptian hieroglyphics | Rosetta Stone (bilingual) |
| Cuneiform | Behistun Inscription (trilingual) |
| Linear B | Known language (Greek) |
| Indus | None of these advantages |

## Characteristics

### Sign Inventory

| Estimate | Count |
|----------|-------|
| Conservative | ~400 signs |
| Liberal | ~600+ signs |
| Core signs | ~200 common |
| Rare signs | Many used once |

### What This Suggests

| Sign count | Implication |
|------------|-------------|
| Too few for logographic | Like Chinese (~5,000+) |
| Too many for alphabet | Like English (26) |
| Probable | Logo-syllabic (like cuneiform) |

### Text Direction

| Evidence | Conclusion |
|----------|------------|
| Sign cramping | Right edge cramped |
| Seal impressions | Mirror image |
| Likely | Right to left |

### Text Length

| Length | Percentage |
|--------|------------|
| 1-5 signs | ~75% |
| 6-10 signs | ~20% |
| 11+ signs | ~5% |
| Maximum | ~26 signs |

### The "Not a Script" Controversy

| Argument | Counter |
|----------|---------|
| Too short | Administrative seals are short |
| No long texts | May have written on perishable materials |
| Symbol system | Combinatorial patterns suggest language |

### Current Consensus

| View | Status |
|------|--------|
| Is writing | Probably yes |
| Language | Unknown (possibly Dravidian) |

## Economic Implications of Undecipherment

### What We Can't Know

| Question | Problem |
|----------|---------|
| Who ruled? | No named rulers |
| Economic organization | No administrative records |
| Prices, wages | No economic texts |
| Property rights | No contracts |
| Trade agreements | No treaties |

### What We Can Infer

| Evidence | Inference |
|----------|-----------|
| Seals widespread | Writing used in trade |
| Standardized signs | Shared system |
| Multiple cities | Communication need |
| Short texts | Perhaps names/titles |

### The Mystery

| Observation | Question |
|-------------|----------|
| Sophisticated economy | How coordinated without readable records? |
| Standardization | How enforced without visible bureaucracy? |
| Wide distribution | How maintained without texts? |

## After the Indus Script

| Timing | Description |
|--------|-------------|

| Observation | Implication |
|-------------|-------------|

### Why Lost?

| Theory | Possibility |
|--------|-------------|

| Limited use | Not widely needed |
| System collapse | Institutions that used it gone |
| Material | Wrote on perishables |
