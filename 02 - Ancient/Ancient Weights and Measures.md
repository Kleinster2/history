# Ancient Weights and Measures

Standardization was an institutional investment that reduced transaction costs—states that standardized enabled commerce. Before reliable weights and measures, every transaction required negotiation over the measuring device itself. Ancient civilizations that established and enforced standards created the infrastructure for trade beyond personal relationships. This was state capacity applied to markets: not commanding what to trade, but enabling any trade to occur.

## Mesopotamian Standards

### The Weight System

| Unit | Approximate Value |
|------|-------------------|
| Shekel | ~8.3 grams |
| Mina | 60 shekels (~500 grams) |
| Talent | 60 minas (~30 kg) |

### Sexagesimal System

| Feature | Description |
|---------|-------------|
| Base | 60 (divisible by 2,3,4,5,6,10,12,15,20,30) |
| Origin | Mesopotamian mathematics |
| Persistence | Time (60 min/hr), geometry (360°) |
| Advantage | Easy fractional calculation |

### Institutional Enforcement

| Mechanism | Function |
|-----------|----------|
| Temple standards | Maintained reference weights |
| Palace verification | Official calibration |
| Clay tablets | Documented transactions |
| Merchant self-regulation | Kārum oversight |
| Penalties | Fraud punishment |

### Economic Effect

Standardized weights enabled grain, silver, and textile markets across [[Mesopotamia|Mesopotamian]] city-states. A merchant in Assur could trust a shekel measured in Babylon. This wasn't automatic—it required institutional infrastructure: official standards, verification, and punishment for fraud.

## Egyptian Measures

### State Standardization

| Domain | Standard |
|--------|----------|
| Length | Royal cubit (~52.4 cm) |
| Weight | Deben (~91 grams) |
| Volume | Hekat (grain measure, ~4.8 liters) |
| Area | Aroura (land measure) |

### Cubit Rods

| Feature | Description |
|---------|-------------|
| Material | Wood, stone, metal |
| Marking | Subdivisions in palms, fingers |
| Distribution | Officials, architects, craftsmen |
| Calibration | Royal standard maintained |

### Flood Measurement

| Function | Importance |
|----------|------------|
| Nilometer | Measured annual flood |
| Tax basis | Flood height → expected yield → tax |
| Prediction | Historical records enabled forecasting |
| State capacity | Information for administration |

### Economic Function

Egyptian standards served state administration more than commerce. Precise land measurement enabled taxation after each flood. Building standards ensured monumental architecture. But commerce remained secondary to [[Ancient Egypt|redistribution]]—standards served the palace, not the market.

## Chinese Unification

### Qin Standardization (221 BCE)

| Domain | Reform |
|--------|--------|
| Weights | Unified across empire |
| Measures | Standard volume, length |
| Currency | Standard coins |
| Axle width | Uniform cart gauge |
| Writing | Standardized script |

### Why Qin Standardization?

| Factor | Explanation |
|--------|-------------|
| Conquest | Unified previously warring states |
| Administration | Needed common measures for taxation |
| Trade | Reduced transaction costs |
| Control | Standards = state power |

### Implementation

| Mechanism | Method |
|-----------|--------|
| Official standards | Distributed bronze measures |
| Inscriptions | Marked with emperor's edict |
| Enforcement | Severe penalties |
| Legacy | Han and later dynasties maintained |

### Economic Effect

Qin standardization created the world's largest unified market. A grain measure in Canton meant the same as in Xi'an. This was [[State Capacity|state capacity]] enabling commerce—not through command but through infrastructure. The standards survived dynasties, becoming permanent institutional capital.

## The Deeper Point

Standardization was an institutional investment that reduced transaction costs—[[State Capacity|states that standardized]] enabled commerce. [[Mesopotamia|Mesopotamian]] temples maintained reference weights that made silver usable as [[Money|money]]. Chinese emperors unified measures across an empire, creating the world's largest integrated market. Roman standards spread with conquest, enabling Mediterranean-wide trade. But standards required ongoing enforcement—when states weakened, measures fragmented, and transaction costs rose. The medieval period would face this challenge: rebuilding standards region by region, fair by fair, guild by guild. What ancient standardization reveals is that markets are not natural—they require institutional infrastructure. Prices mean nothing if measures vary. Contracts are unenforceable if terms are ambiguous. Long-distance trade is impossible if every transaction requires renegotiating units. [[Institutions and Progress|States that provided this infrastructure]] enabled commerce; those that didn't confined trade to local, personal exchange.
