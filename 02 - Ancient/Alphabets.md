# Alphabets

The alphabet was one of humanity's great simplifications. Where cuneiform needed hundreds of signs and Chinese thousands, the alphabet reduced writing to 20-30 symbols representing sounds. This democratized literacy, spread along trade routes, and became the basis for most of the world's writing systems. A Phoenician merchant innovation became the foundation of Greek, Latin, Arabic, Hebrew, and countless other scripts.

## Overview

### What Makes an Alphabet

| Feature | Description |
|---------|-------------|
| Principle | One sign = one sound (phoneme) |
| Size | 20-30 symbols (vs. hundreds/thousands) |
| Learning | Easier than logographic systems |
| Flexibility | Can write any word |

### Before Alphabets

| System | Signs | Type |
|--------|-------|------|
| Cuneiform | 600+ | Syllabic + logographic |
| Hieroglyphics | 700+ | Logographic + phonetic |
| Linear B | 90+ | Syllabic |
| Chinese | Thousands | Logographic |

## The Greek Adaptation

### The Crucial Innovation: Vowels

| Phoenician | Greek Use |
|------------|-----------|
| Consonants only | Added vowels |
| Aleph (glottal stop) | Alpha (a) |
| He (h) | Epsilon (e) |
| Yodh (y) | Iota (i) |
| Ayin (pharyngeal) | Omicron (o) |

**Greeks repurposed consonants their language didn't need as vowels‚Äîcreating the first true alphabet.**

### Why It Mattered

| Feature | Effect |
|---------|--------|
| Complete representation | Every sound written |
| Unambiguous | No guessing vowels |
| Learning | Even easier |
| Literature | Enabled Homer to be written down |
