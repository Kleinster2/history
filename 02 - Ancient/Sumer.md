# Sumer

Sumer, located in southern Mesopotamia (modern southern Iraq), is widely considered the world's first civilization. The Sumerians developed writing, built the first cities, created sophisticated religious and political institutions, and established cultural patterns that influenced the entire ancient Near East for millennia.

## Economy

### Economic Overview

| Feature | Description |
|---------|-------------|
| Type | Mixed: temple, palace, private |
| Currency | Silver by weight (no coins) |
| Standard unit | Shekel (~8.3 grams silver) |
| Recordkeeping | Writing invented for accounts |
| Innovation | First market economy |

### Agriculture

| Aspect | Details |
|--------|---------|
| Staples | Barley, dates, vegetables |
| Irrigation | Canal systems (state-managed) |
| Plow | Animal-drawn (oxen) |
| Labor | Temple workers, private farmers |
| Storage | Temple and palace granaries |
| Yields | 20:1 to 80:1 grain ratios |

### Temple Economy

| Function | Description |
|----------|-------------|
| Land ownership | Temples owned 20-30% of land |
| Employment | Thousands of dependents |
| Redistribution | Rations to workers |
| Workshops | Textile, metalwork production |
| Trade | Surplus exchange |

### Temple Workforce

| Category | Rations |
|----------|---------|
| Male workers | 60 liters barley/month |
| Female workers | 30 liters barley/month |
| Children | 15-20 liters/month |
| Supervisors | Higher rations + other goods |

### Ur III Economy (2112-2004 BCE)

| Feature | Scale |
|---------|-------|
| Bureaucracy | Most documented ancient economy |

| Workshops | State-run textile factories |
| Workers | Thousands in state employ |

### Prices and Wages

|------|-------------------|

| 1 sheep | 1 shekel |

| 1 ox | 10-15 shekels |
