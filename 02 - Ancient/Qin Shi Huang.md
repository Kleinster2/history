# Qin Shi Huang

Qin Shi Huang (259-210 BCE), the "First Emperor," unified China and created the template for Chinese imperial economy. His standardization of currency, weights, measures, and writing enabled economic integration across a vast territoryâ€”and his massive projects demonstrated both the power and the costs of state-directed economy.

## Economic Unification

### The Problem He Faced

| Issue | Description |
|-------|-------------|
| Seven currencies | Incompatible money |
| Multiple measure systems | Trade barriers |
| Different scripts | Communication difficulty |
| Regional customs | Economic fragmentation |

### His Solution: Standardization

| Standard | Implementation |
|----------|----------------|
| Currency | Banliang coin empire-wide |
| Weights | Unified jin, liang |
| Measures | Standard chi, sheng, dou |
| Script | Small seal script |
| Axle width | Uniform for roads |

### Economic Logic

| Action | Purpose |
|--------|---------|
| Single currency | One market |
| Standard measures | Fair taxation |
| Common script | Administrative efficiency |
| Same axle widths | Road system |

## Legacy

### What Survived

| Institution | Duration |
|-------------|----------|
| Round coin | 2,100+ years |
| Commandery system | Modified, continued |
| Unified weights/measures | Periodically updated |
| Imperial model | Until 1912 |

### Lessons for Economic History

| Lesson | Interpretation |
|--------|----------------|
| Standardization enables | Markets require common measures |
| States can transform | But at what cost? |
| Extraction has limits | Populations rebel |
| Infrastructure persists | Long-term investment value |

### Han Reaction

| Qin Excess | Han Modification |
|------------|------------------|
| Heavy taxes | Initially lightened |
| Harsh corvee | Somewhat reduced |
| Legalism only | Confucianism incorporated |
| Anti-commerce | More pragmatic |
