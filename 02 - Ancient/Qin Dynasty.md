# Qin Dynasty

The Qin dynasty (221-206 BCE) lasted only fifteen years but transformed Chinese economic history forever. Through brutal efficiency and systematic standardization, the First Emperor unified weights, measures, currency, and even axle widths—creating the template for Chinese economic administration for the next two millennia.

## Overview

| Aspect | Details |
|--------|---------|
| Period | 221-206 BCE (15 years) |
| Founder | Qin Shi Huang (First Emperor) |

## Economic Standardization

### The Great Unification

| Standard | Before | After |
|----------|--------|-------|
| Coins | Knife, spade, cowrie, etc. | Round banliang coin |
| Weights | Regional systems | Unified system |
| Measures | Variable | Standard lengths |
| Axle widths | Different | Same (for roads) |
| Script | Regional variants | Small seal script |

### Why Standardization Mattered

| Benefit | Description |
|---------|-------------|
| Trade | Easier exchange across empire |
| Taxation | Consistent assessment |
| Administration | Uniform records |
| Military | Interchangeable equipment |
| Control | State oversight |

## The Banliang Coin

### Specifications

| Feature | Details |
|---------|---------|
| Shape | Round with square hole |
| Weight | Ban liang (~8 grams) |
| Material | Bronze |
| Inscription | 半兩 (ban liang = "half liang") |
| Significance | Template for 2,000 years |

### Replacing Diversity

| Previous Type | State |
|---------------|-------|
| Knife money | Qi, Yan |
| Spade money | Various Zhou states |
| Ant-nose | Chu |
| Cowrie-shaped | Chu |
| Round coins | Qin (now universal) |

### Economic Impact

| Effect | Description |
|--------|-------------|
| Single currency | Unified market |
| State monopoly | Controlled minting |
| Eliminated confusion | Standard value |
| Control | Economic sovereignty |
