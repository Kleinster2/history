# American Revolution

The American Revolution was an economic event as much as a political one. Britain's richest and most commercially developed colonies rebelled not from poverty but from prosperityâ€”and from resistance to imperial policies that threatened colonial economic autonomy. The revolution's outcome created a new nation facing the challenge of building economic institutions from scratch.

## Overview

### The Paradox

| Factor | Reality |
|--------|---------|
| Colonial wealth | Among highest living standards in world |
| Growth rate | Rapid, outpacing Britain |
| Trade | Thriving, despite restrictions |
| Yet | Rebellion against the empire |

### Timeline

| Period | Development |
|--------|-------------|
| 1763 | French and Indian War ends; Britain seeks revenue |
| 1765 | Stamp Act crisis |
| 1767 | Townshend Acts |
| 1773 | Tea Act, Boston Tea Party |
| 1775-1783 | War |
| 1783 | Independence recognized |

---

## Economic Causes

### The Debt Problem

| Factor | Description |
|--------|-------------|
| War costs | French and Indian War doubled British debt |
| Defense costs | 10,000 troops stationed in America |
| British view | Colonies should pay for their defense |
| Colonial view | Already contributing through trade |

### Trade Restrictions

| Policy | Effect |
|--------|--------|
| Navigation Acts | Trade through Britain only |
| Enumerated goods | Key exports to Britain only |
| Manufacturing limits | Colonies import British goods |
| Enforcement | Lax until 1763 |

### The Shift After 1763

| Before 1763 | After 1763 |
|-------------|------------|
| Salutary neglect | Active enforcement |
| Smuggling tolerated | Customs crackdown |
| Local autonomy | Imperial control |
| Low taxes | New revenue measures |

---

## The Tax Disputes

### Stamp Act (1765)

| Feature | Description |
|---------|-------------|
| What | Tax on paper, documents, playing cards |
| Amount | Relatively small |
| Issue | Internal tax without consent |
| Response | Riots, boycotts, repeal (1766) |

### Townshend Acts (1767)

| Feature | Description |
|---------|-------------|
| What | Duties on glass, lead, paper, tea |
| Revenue | Used to pay royal governors |
| Issue | Undermined colonial assemblies |
| Response | Non-importation agreements |

### Tea Act (1773)

| Feature | Description |
|---------|-------------|
| What | [[East India Companies\|East India Company]] monopoly on tea |
| Price | Actually lowered tea prices |
| Issue | Principle of taxation, monopoly |
| Response | Boston Tea Party |

### The Real Issue

| British View | Colonial View |
|--------------|---------------|
| Parliament sovereign | No taxation without representation |
| Virtual representation | Only local assemblies can tax |
| Empire needs revenue | Charter rights violated |

## Economic Warfare

### Colonial Boycotts

| Period | Action |
|--------|--------|
| 1765-1766 | Non-importation (Stamp Act) |
| 1768-1770 | Non-importation (Townshend) |
| 1774-1775 | Continental Association |

### Effectiveness

| Factor | Result |
|--------|--------|
| British exports to colonies | Dropped 40% (1768-1769) |
| Colonial unity | Strengthened |
| British merchants | Lobbied for repeal |
| Limitations | Hard to sustain |

### British Coercion

| Measure | Effect |
|---------|--------|
| Boston Port Act (1774) | Closed Boston harbor |
| Economic pressure | Intended to isolate Massachusetts |
| Result | United colonies instead |

## Wartime Economy

### Financing the War

| Method | Description |
|--------|-------------|
| Paper money | Continental dollars |
| Loans | Foreign (France, Netherlands) |
| Domestic bonds | Certificates, IOUs |
| Requisitions | States asked to contribute |

### The Continental Dollar

| Period | Value |
|--------|-------|
| 1775 | Par |
| 1778 | 4:1 |
| 1780 | 40:1 |
| 1781 | Worthless ("not worth a Continental") |

### War Costs

| Factor | Estimate |
|--------|----------|
| Total cost | $400+ million (in specie) |
| Debt at end | ~$40 million foreign, ~$40 million domestic |
| State debts | ~$25 million |
| How paid | Inflation, default, restructuring |

## Economic Impacts

### Winners

| Group | Gain |
|-------|------|
| Merchants | Free trade, new markets |
| Land speculators | Western lands opened |
| Artisans | Protected from British competition |
| Debtors | Inflation reduced real debts |

### Losers

| Group | Loss |
|-------|------|
| Loyalists | Property confiscated |
| Creditors | Paid in depreciated currency |
| British merchants | Lost colonial markets |
| Enslaved people | Revolution didn't free them |

### Structural Changes

| Before | After |
|--------|-------|
| Trade restricted to Britain | Free trade (in theory) |
| British credit networks | Disrupted |
| Imperial protection | Gone |
| Stable currency | Monetary chaos |

## The Critical Period (1783-1789)

### Economic Problems

| Problem | Description |
|---------|-------------|
| Debt | War debts unpaid |
| Currency | No national money |
| Trade | British markets closed |
| Revenue | Congress couldn't tax |

### The Depression

| Factor | Description |
|--------|-------------|
| British goods | Flooded markets |
| Specie drain | Hard money flowed out |
| Farm prices | Collapsed |
| Debtors | Couldn't pay |

### Shays' Rebellion (1786-1787)

| Factor | Description |
|---------|-------------|
| Cause | Farm foreclosures, debt courts |
| Location | Western Massachusetts |
| Response | State militia suppressed |
| Effect | Showed weakness of Confederation |

## The Constitutional Solution

### Economic Provisions

| Clause | Purpose |
|--------|---------|
| Commerce clause | Congress regulates interstate trade |
| Tax power | Congress can levy taxes |
| Uniform duties | No state tariffs |
| Contract clause | States can't impair contracts |
| Sound money | States can't print money |

### What It Enabled

| Power | Use |
|-------|-----|
| Tariffs | Revenue + protection |
| Debt assumption | [[Hamilton's Financial System]] |
| National bank | Credit and currency |
| Treaties | Trade agreements |

## Historical Interpretations

|----------------|----------|
| Progressive (Beard) | Economic interests drove founders |

| Argument | Evidence |
|----------|----------|

| Long-term gains | Free trade, expansion, institutions |
