# Islamic Economy

The Islamic world created the largest integrated economic zone of the medieval period, stretching from Spain to Central Asia. Islamic civilization pioneered financial instruments, agricultural techniques, and commercial practices that would later spread to Europe—challenging narratives of medieval economic stagnation.

## Overview

| Aspect | Details |
|--------|---------|
| Period | 622-1500 CE (focus on 750-1250) |
| Geographic extent | Iberia to Central Asia, Sahara to Caucasus |
| Major centers | Baghdad, Cairo, Córdoba, Damascus |
| Peak GDP share | ~20-25% of world |
| Key innovations | Checks, partnerships, crop rotation |

## Agricultural Revolution

### New Crops (7th-11th c.)

The Islamic agricultural revolution introduced crops from India and elsewhere across the Mediterranean.

| Crop | Origin | Impact |
|------|--------|--------|
| Sugar cane | India | Major industry |
| Cotton | India | Textile revolution |
| Rice | Asia | New staple |
| Citrus fruits | Southeast Asia | New products |
| Spinach | Persia | Diet variety |
| Eggplant | India | Diet variety |

| Innovation | Effect |
|------------|--------|
| Irrigation expansion | Extended cultivation |
| Crop rotation | Better yields |
| Summer cropping | Multiple harvests |

## Money and Finance

### Currency

| Currency | Description |
|----------|-------------|
| Dinar (gold) | Standard gold coin |
| Dirham (silver) | Standard silver coin |
| Exchange rates | Fluctuated by supply |
| Minting | Caliphal and regional |

### Financial Innovations

| Innovation | Description | Later Transmission |
|------------|-------------|-------------------|
| Sakk (check) | Written payment order | → European "check" |
| Suftaja | Bill of exchange | → European bills |
| Hawala | Value transfer system | Still used |
| Mudaraba | Profit-sharing partnership | → Commenda |
| Musharaka | Joint venture | Partnership forms |
| Qirad | Investment contract | → Societas |

### Islamic Finance Principles

| Principle | Meaning | Effect |
|-----------|---------|--------|
| Riba prohibition | No interest | Alternative structures |
| Gharar prohibition | No excessive uncertainty | Contract clarity |
| Profit/loss sharing | Risk distribution | Partnership emphasis |
| Asset-backing | Real economy link | Less speculation |

### How Interest Prohibition Worked

| Need | Islamic Solution |
|------|-----------------|
| Consumer lending | Deferred sale (murabaha) |
| Investment capital | Profit-sharing (mudaraba) |
| Partnership | Joint venture (musharaka) |
| Real estate | Sale-leaseback |
| Working capital | Deferred payment |

**In practice:** These structures often achieved similar economic functions to interest-based lending while maintaining formal compliance.

## Decline and Legacy

### Relative Decline (13th-15th c.)

| Factor | Effect |
|--------|--------|
| Mongol invasions | Devastation (Baghdad 1258) |
| Black Death | Population loss |
| Trade route shifts | Less centrality |
| Political fragmentation | Economic disruption |
| New powers | Ottomans, Mamluks different |

### Why Not Industrial Revolution?

| Factor | Argument |
|--------|----------|
| Waqf rigidity | Property locked up |
| Labor costs | Not scarce enough |
| Political fragmentation | No unified market |
| Mongol devastation | Destroyed infrastructure |
| Geographic position | Bypassed by Atlantic trade |

**Note:** This is debated. Islamic world remained wealthy; question is why no industrialization.

### Legacy to Europe

| Transfer | Impact |
|----------|--------|
| Financial instruments | Banking development |
| Agricultural crops | European agriculture |
| Paper production | Information revolution |
| Numerals | Mathematics |
| Scientific knowledge | Renaissance foundations |
| Trade networks | European participation |
