# Florence

Florence revolutionized European finance, creating the banking system that funded popes, kings, and the Renaissance itself. Unlike maritime Venice, Florence built its wealth on manufacturing (wool finishing) and banking—innovating double-entry bookkeeping, international banking networks, and the gold florin that became Europe's standard currency.

|--------|---------|

## The Banking Revolution

### Why Florence Led

| Factor | Advantage |
|--------|-----------|
| Wool trade | Generated capital |
| Pope connections | Papal banking |
| Florin | Trusted currency |
| Innovation | New techniques |

### The Florin

| Feature | Description |
|---------|-------------|
| Introduced | 1252 |
| Metal | Gold (3.5 grams) |
| Purity | High, consistent |
| Acceptance | International standard |
| Duration | Used for centuries |
| Significance | First major gold coin since Rome |

### Banking Innovations

| Innovation | Description |
|------------|-------------|

| Double-entry bookkeeping | Systematic accounting |
| Bills of exchange | Credit transfer |
| Letters of credit | Trade finance |
| Deposit banking | Interest-bearing accounts |

## The Bardi-Peruzzi Crash (1340s)

### The Situation

| Factor | Description |
|--------|-------------|
| Scale | Largest banks in Europe |
| Clients | Kings, popes, merchants |
| Loans | Massive to Edward III |
| Risk | Concentrated exposure |

### The Collapse

| Event | Impact |
|-------|--------|
| Edward III default | Couldn't repay loans |
| Hundred Years War | English couldn't pay |
| Bardi bankruptcy | 1343 |
| Peruzzi bankruptcy | 1343 |
| Economic crisis | European depression |

### Lessons Learned

| Lesson | Later Practice |
|--------|----------------|
| Sovereign risk | Diversify clients |
| Concentration | Spread risk |
| Collateral | Require security |
| Political exposure | Cautious royal lending |

### Economic Governance

| Function | Method |
|----------|--------|
| Guild regulation | Quality standards |
| Tax collection | Catasto (property tax) |
| Debt management | Monte Comune |
| Currency | Mint control |

### The Catasto (1427)

| Feature | Description |
|---------|-------------|
| Innovation | Wealth-based tax |
| Method | Asset declaration |
| Assessment | Progressive rates |
| Records | Detailed wealth data |
| Significance | First systematic wealth tax |

Recent research using the 1427 Catasto shows remarkable wealth persistence—descendants of wealthy families in 1427 remain wealthier today, 600 years later.

**See:** [[Economic Persistence]] for the Barone-Mocetti study.
