# Cairo

Cairo was the greatest city of the medieval Islamic world after Baghdad's declineâ€”a commercial hub connecting the Mediterranean to the Indian Ocean, a center of learning and culture, and seat of successive powerful dynasties. For centuries, Cairo dominated East-West trade, and its wealth funded monuments that still define Egypt. The city's history spans Fatimid splendor, Mamluk power, and Ottoman incorporation.

## Overview

| Feature | Description |
|---------|-------------|
| Founded | 969 CE (Fatimid) |
| Earlier settlements | Fustat (641), al-Qata'i |
| Peak population | ~500,000 (14th century) |
| Location | Nile delta, strategic position |

### Successive Rulers

| Dynasty | Period | Character |
|---------|--------|-----------|
| Fatimid | 969-1171 | Founding, Shi'i caliphate |
| Ayyubid | 1171-1250 | Saladin, Sunni restoration |
| Mamluk | 1250-1517 | Military elite, trade peak |
| Ottoman | 1517-1798 | Provincial capital |

|---------|-------------|

## Mamluk Cairo (1250-1517)

### The Mamluk System

| Feature | Description |
|---------|-------------|
| Rulers | Military slaves, became elite |
| Succession | Non-hereditary (mostly) |
| Economy | State-controlled trade |
| Culture | Patronage, monuments |

### Commercial Hub

| Trade Route | Connection |
|-------------|------------|
| Red Sea | India, spices, textiles |
| Mediterranean | Venice, Genoa, Europe |
| Nile | Interior Africa |
| Overland | Syria, Iraq |

### The Karimi Merchants

| Feature | Description |
|---------|-------------|
| Identity | Elite spice traders |
| Operations | Red Sea-Mediterranean |
| Wealth | Immense fortunes |
| Decline | Portuguese circumnavigation |

### Mamluk Economy

| Sector | Description |
|--------|-------------|

| Agriculture | Nile valley |
| Textiles | Cotton, linen |
| Sugar | Major producer |

|---------|-------------|

| Guilds | Organized trades |
