# Mongol Economy

The Mongol Empire created the largest contiguous land empire in historyâ€”and paradoxically, history's greatest free trade zone. The Pax Mongolica connected China to Europe under a single political authority, enabling unprecedented movement of goods, people, ideas, and diseases across Eurasia. The Mongols were destroyers and connectors simultaneously.

## Overview

| Aspect | Details |
|--------|---------|
| Period | 1206-1368 (unified/fragmented) |
| Extent | Pacific to Eastern Europe |
| Population | ~100 million governed |
| Founder | Genghis Khan |
| Economic role | Trade facilitators |
| Legacy | Pax Mongolica |

## The Pax Mongolica

### What It Was

| Feature | Description |
|---------|-------------|
| Period | ~1250-1350 |
| Meaning | "Mongol Peace" |
| Effect | Safe Eurasian travel |
| Trade | Boomed |
| Exchange | Unprecedented |

### How It Worked

| Mechanism | Description |
|-----------|-------------|
| Single authority | One empire, one law |
| Yam system | Postal relay stations |
| Protection | Caravans guarded |
| Merchants favored | Mongol policy |
| Passports (paiza) | Safe conduct |

### The Yam System

| Feature | Description |
|---------|-------------|
| Stations | Every 25-50 km |
| Horses | Fresh mounts |
| Riders | Official messengers |
| Speed | Fastest communication |
| Secondary use | Trade facilitation |
