# Bipedalism

Bipedalism—habitual upright walking on two legs—is the defining adaptation of the hominin lineage. Evolving approximately 7-6 million years ago, bipedalism preceded large brains, tool use, and language by millions of years, making it the foundational trait that set our ancestors apart from other apes.

## Overview

### Definition

| Term | Description |
|------|-------------|
| Bipedalism | Locomotion on two legs |
| Obligate biped | Species that must walk bipedally (humans) |
| Facultative biped | Can walk bipedally but uses other modes (chimps) |
| Striding gait | Human-style walking with extended leg |

### Uniqueness

| Feature | Humans | Other Bipeds |
|---------|--------|--------------|
| Habitual | Yes | No (among mammals) |
| Striding | Yes | No |
| Endurance | Exceptional | Limited |
| Energy efficiency | High | Variable |

## Timeline of Bipedal Evolution

| Time (Ma) | Species | Evidence | Bipedal Status |
|-----------|---------|----------|----------------|
| 7-6 | *Sahelanthropus* | Skull base | Possibly bipedal |
| 6 | *Orrorin* | Femur | Bipedal + climbing |
| 5.5-4.4 | *Ardipithecus* | Pelvis, foot | Facultative biped |
| 4.2-3.0 | *Australopithecus* | Full skeleton | Obligate biped |
| 3.66 | Laetoli footprints | Trackway | Modern gait |
| 2.0-1.5 | *Homo erectus* | Full skeleton | Fully modern |

## Anatomical Adaptations

### Skull and Spine

| Feature | Ape | Bipedal Hominin |
|---------|-----|-----------------|
| Foramen magnum | Posterior | Inferior (under skull) |
| Spine | Single curve | S-shaped (lordosis) |
| Head balance | Muscles support | Balanced on spine |
| Nuchal muscles | Large | Reduced |

### Pelvis

| Feature | Ape | Human |
|---------|-----|-------|
| Ilium (hip blade) | Tall, flat | Short, curved bowl |
| Orientation | Backward | Sideways |
| Gluteal muscles | Extensors | Abductors (balance) |
| Birth canal | Simple | Complex, rotational |
| Sacrum | Narrow | Wide |

### Legs

| Feature | Ape | Human |
|---------|-----|-------|
| Femur angle | Straight | Valgus (angled inward) |
| Knee | Simple hinge | Locking mechanism |
| Leg length | Short relative | Long relative |
| Thigh muscles | Climbing | Walking/running |

### Foot

| Feature | Ape | Human |
|---------|-----|-------|
| Big toe | Opposable, divergent | Aligned, non-grasping |
| Arch | Absent | Longitudinal and transverse |
| Toes | Long, curved | Short, straight |
| Function | Grasping | Platform, propulsion |

### Arms and Shoulders

| Feature | Ape | Human |
|---------|-----|-------|
| Arm length | Long | Shorter |
| Shoulder | Upward-oriented | Lateral |
| Carrying angle | Climbing | Freed for carrying |

## Why Did Bipedalism Evolve?

### Hypotheses

| Hypothesis | Mechanism | Evidence |
|------------|-----------|----------|
| **Postural feeding** | Reaching fruit in trees | Ardi's woodland habitat |
| **Thermoregulation** | Reduce sun exposure | Open habitat advantages |
| **Freeing hands** | Carrying food/tools/infants | Universal in hominins |
| **Energy efficiency** | Walking cheaper than quadrupedalism | Biomechanical studies |
| **Vigilance** | See over tall grass | Predator detection |
| **Provisioning** | Males carry food to females | Canine reduction correlation |
| **Wading** | Bipedal in water | Aquatic ape (largely rejected) |

### Current Understanding

No single cause; likely multiple factors:
- Began in woodland/forest context (Ardi)
- Postural feeding may have initiated
- Benefits accumulated (carrying, efficiency, thermoregulation)
- Open habitats reinforced advantages later

## Costs of Bipedalism

### Trade-offs

| Cost | Description |
|------|-------------|
| Slower than quadrupeds | Can't outrun predators in sprint |
| Birth complications | Narrow pelvis vs. large-brained babies |
| Back problems | Lordosis causes lower back issues |
| Knee/hip stress | Weight-bearing on two legs |
| Varicose veins | Blood pooling in legs |
| Hernias | Abdominal pressure |

### The Obstetric Dilemma

| Conflict | Description |
|----------|-------------|
| Pelvis for walking | Needs to be narrow |
| Pelvis for birth | Needs to be wide |
| Baby brain size | Keeps increasing |
| Solution | Babies born "early," rotational birth |

## Biomechanics

### Energy Efficiency

| Mode | Energy Cost |
|------|-------------|
| Human bipedal walking | Baseline (1.0) |
| Chimp bipedal walking | ~2x more expensive |
| Chimp quadrupedal | Similar to human bipedal |
| Human running | Less efficient than walking |

### The Striding Gait

| Phase | Description |
|-------|-------------|
| Heel strike | Foot contacts ground heel-first |
| Stance phase | Weight transfers over foot |
| Toe-off | Push off from big toe |
| Swing phase | Leg moves forward |
| Pendulum action | Energy-efficient leg swing |

### Endurance Running

| Feature | Advantage |
|---------|-----------|
| Sweating | Cooling during exertion |
| Springy tendons | Energy storage/return |
| Gluteus maximus | Running stabilization |
| Nuchal ligament | Head stabilization |
| Short toes | Efficient push-off |

## Bipedalism Through Time

### Early Bipeds (7-4 Ma)

| Species | Locomotion |
|---------|------------|
| *Sahelanthropus* | Unknown, possibly bipedal |
| *Orrorin* | Bipedal walking + tree climbing |
