# Early Homo

The genus *Homo* emerged approximately 2.8-2.5 million years ago, marking a major transition in human evolution. Early *Homo* species were characterized by larger brains, reduced dental size, increased meat consumption, and stone tool manufacture - adaptations that would eventually lead to global dispersal and modern humans.

## Overview

### Defining the Genus Homo

What makes a hominin *Homo* rather than *Australopithecus*?

| Feature | *Australopithecus* | Early *Homo* |
|---------|-------------------|--------------|
| Brain size | 400-550 cc | 500-900 cc |
| Teeth | Large molars | Reduced molars |
| Face | Prognathic | Less projecting |
| Body proportions | Long arms, short legs | More modern |
| Diet | Plant-focused | More omnivorous |
| Tools | Rare/absent | Consistent makers |

### Early Homo Species

| Species | Age (Ma) | Location | Brain Size (cc) |
|---------|----------|----------|-----------------|
| *H. habilis* | 2.4-1.4 | East Africa | 500-700 |
| *H. rudolfensis* | 2.0-1.8 | East Africa | 700-800 |
| *H. ergaster* | 1.9-1.4 | Africa | 700-850 |
| *H. erectus* | 1.9-0.1 | Africa, Asia, Europe | 800-1250 |
| *H. naledi* | 0.34-0.24 | South Africa | 465-560 |
| *H. georgicus* | 1.85-1.75 | Georgia | 600-775 |

## Homo habilis (2.4-1.4 Ma)

"Handy Man" - the first species assigned to our genus.

### Discovery
- First specimens: 1960-1963, Olduvai Gorge, Tanzania
- Described by: Louis Leakey, Phillip Tobias, John Napier (1964)
- Key specimen: OH 7 (type specimen) - partial skull and hand

### Key Specimens

| Specimen | Site | Features |
|----------|------|----------|
| OH 7 | Olduvai | Type specimen, mandible and parietals |
| OH 24 ("Twiggy") | Olduvai | Crushed cranium |
| OH 62 | Olduvai | Partial skeleton |
| KNM-ER 1813 | Koobi Fora | Complete small cranium |

### Anatomy

| Feature | Measurement/Condition |
|---------|----------------------|
| Brain size | 500-700 cc |
| Body size | ~1.0-1.3 m tall, 30-35 kg |
| Face | Projecting but reduced from australopiths |
| Teeth | Smaller than australopiths |
| Arms | Long relative to legs |
| Hands | Precision grip capable |

### Tool Association
*H. habilis* is associated with Oldowan stone tools:
- Simple flake tools and cores
- Used for processing meat and plants
- Found at same sites as fossils

### Debate: One Species or Two?
The *habilis* sample is highly variable. Some researchers split it into:
- *H. habilis* sensu stricto (smaller brained)
- *H. rudolfensis* (larger brained)

---

## Homo rudolfensis (2.0-1.8 Ma)

A larger-brained contemporary of *H. habilis*.

### Discovery
- Key specimen: KNM-ER 1470
- Found: 1972, Koobi Fora, Kenya
- Discovered by: Bernard Ngeneo (Richard Leakey's team)

### Anatomy

| Feature | Compared to *H. habilis* |
|---------|-------------------------|
| Brain size | 700-800 cc (larger) |
| Face | Flatter, larger |
| Teeth | Larger molars |
| Build | Possibly larger overall |

### Taxonomic Controversy
Status debated:
- Valid species distinct from *habilis*?
- Large males of *habilis*?
- Recently discovered fossils support species distinction

---

## Homo ergaster (1.9-1.4 Ma)

"Working Man" - the African form often linked to *H. erectus*.

### Discovery and Naming
- Some researchers consider this the African population of *H. erectus*
- Others treat it as a distinct species
- Key specimen: KNM-ER 3733

### Key Specimens

| Specimen | Age | Site | Significance |
|----------|-----|------|--------------|
| KNM-ER 3733 | 1.78 Ma | Koobi Fora | Complete adult cranium |
| KNM-WT 15000 ("Turkana Boy") | 1.5 Ma | Nariokotome | Nearly complete skeleton |
| KNM-ER 3883 | 1.5 Ma | Koobi Fora | Cranium |

### Turkana Boy (KNM-WT 15000)

The most complete early *Homo* skeleton:

| Feature | Measurement |
|---------|-------------|
| Age at death | ~8-11 years (juvenile) |
| Height | 1.6 m (would have reached 1.85 m as adult) |
| Brain size | 880 cc (would have reached ~910 cc) |
| Body proportions | Modern human-like |
| Limbs | Long legs, shorter arms |

**Significance:**
- First hominin with fully modern body proportions
- Adapted for long-distance walking/running
- Tropical body build (tall, linear)
- Evidence of developmental patterns

### Anatomy

| Feature | Condition |
|---------|-----------|
| Brain size | 700-850 cc |
| Body size | 1.5-1.85 m tall |
| Body proportions | Modern (long legs, shorter arms) |
| Nose | External nose projection |
| Skin | Likely hairless, sweating |
| Thorax | Barrel-shaped |

## Homo erectus (1.9 Ma - 100 ka)

"Upright Man" - the first hominin to leave Africa and persist for nearly 2 million years.

### Discovery
- First fossils: 1891, Trinil, Java
- Discovered by: Eugène Dubois
- Originally named: *Pithecanthropus erectus* ("Java Man")

### Geographic Range

| Region | Time Range | Key Sites |
|--------|------------|-----------|
| Africa | 1.9-1.0 Ma | Olduvai, Koobi Fora |
| Caucasus | 1.85-1.75 Ma | Dmanisi |
| Indonesia | 1.8 Ma-100 ka | Trinil, Sangiran, Ngandong |
| China | 1.7 Ma-300 ka | Zhoukoudian, Lantian |
| Europe | 1.4-0.7 Ma | Atapuerca (Gran Dolina) |

### Key Specimens

| Specimen | Site | Age | Features |
|----------|------|-----|----------|
| Java Man (Trinil 2) | Java | 1.0 Ma | Type specimen |
| Peking Man | Zhoukoudian | 780-230 ka | Multiple individuals |
| Sangiran 17 | Java | 1.3 Ma | Complete cranium |
| OH 9 | Olduvai | 1.5 Ma | Massive African *erectus* |
| Dmanisi skulls | Georgia | 1.8 Ma | Earliest Eurasian |

### Anatomy

| Feature | Early *erectus* | Late *erectus* |
|---------|-----------------|----------------|
| Brain size | 650-900 cc | 850-1250 cc |
| Brow ridge | Prominent | Very prominent |
| Skull shape | Long, low | Long, low |
| Face | Projecting | Projecting |
| Body height | 1.5-1.85 m | Similar |
| Body build | Tropical/linear | Variable |

### Dmanisi: First Europeans

The Dmanisi site (Georgia) provides crucial evidence:

| Specimens | D2700, D2282, D2280, D3444, D4500 |
|-----------|-----------------------------------|
| Age | 1.85-1.75 Ma |
| Brain size | 546-775 cc |
| Significance | Oldest hominins outside Africa |

**Key Insights:**
- Left Africa before brain expansion complete
- Small-brained individuals successful
- Variable population
- Simple Oldowan-like tools

---

## Homo naledi (335-236 ka)

A surprising late survivor with a mix of primitive and derived features.

### Discovery
- Found: 2013-2015, Rising Star Cave, South Africa
- Discovered by: Lee Berger's team

### Anatomy

| Feature | Condition |
|---------|-----------|
| Brain size | 465-560 cc (very small) |
| Body size | 1.4-1.5 m |
| Hands | Modern, tool-using capable |
| Feet | Modern, bipedal |
| Shoulders | Primitive, climbing adapted |
| Pelvis | Primitive features |

### Significance
- Survived until at least 236 ka
- Contemporary with early *H. sapiens*
- Small brain despite late date
- Possible deliberate body disposal

## Key Adaptations

### Brain Expansion

| Species | Brain Size (cc) | Body Size Estimate |
|---------|-----------------|-------------------|
| *A. afarensis* | 400-500 | 30-45 kg |
| *H. habilis* | 500-700 | 30-40 kg |
| *H. erectus* (early) | 700-900 | 50-65 kg |
| *H. erectus* (late) | 900-1250 | 50-65 kg |

**Possible Drivers:**
- Meat eating / dietary quality
- Social complexity
- Tool use feedback
- Climate variability

### Body Proportions

The shift from australopith to *H. ergaster/erectus* body plan:

| Feature | Australopith | *H. ergaster* |
|---------|--------------|---------------|
| Relative leg length | Short | Long |
| Relative arm length | Long | Shorter |
| Hip width | Wide | Narrower |
| Barrel ribcage | No | Yes |
| Body shape | Climbing-adapted | Striding-adapted |

### Dietary Shift

Evidence for increased meat consumption:
- Larger brain requires more calories
- Smaller gut (reduced rib cage)
- Cut marks on animal bones
- Stone tool association
- Isotopic evidence

### Fire Use

| Site | Age | Evidence |
|------|-----|----------|
| Koobi Fora | 1.5 Ma | Discolored sediments (disputed) |
| Wonderwerk Cave | 1.0 Ma | Burned bone and plants |
| Gesher Benot Ya'aqov | 780 ka | Burned materials |
| Zhoukoudian | 400-500 ka | Abundant ash, bone |

Confirmed controlled fire: ~400 ka; possible earlier use uncertain.

## First Dispersal from Africa

### Timing and Routes

*Homo erectus* was the first hominin to leave Africa:

| Corridor | Evidence |
|----------|----------|
| Sinai/Levant | Ubeidiya (Israel), 1.4 Ma |
| Southern Route | Possible coastal route to Java |

### Early Sites Outside Africa

| Site | Age (Ma) | Location |
|------|----------|----------|
| Dmanisi | 1.85-1.75 | Georgia |
| 'Ubeidiya | 1.4 | Israel |
| Sangiran | 1.6-1.0 | Java |
| Yuanmou | 1.7 | China |
| Atapuerca (TD6) | 1.2 | Spain |

### What Enabled Dispersal?
- Improved foraging abilities
- Tool technology (Oldowan, then Acheulean)
- Adaptable diet
- Social cooperation
- Fire use (later)

## Stone Tool Industries

### Oldowan (2.6-1.7 Ma)

| Feature | Description |
|---------|-------------|
| Core tools | Simple choppers |
| Flake tools | Unretouched flakes |
| Raw materials | Local cobbles |
| Reduction | Minimal |
| Associated species | *H. habilis*, early *erectus* |

### Acheulean (1.76 Ma - 200 ka)

| Feature | Description |
|---------|-------------|
| Signature tool | Handaxe (biface) |
| Technique | Bifacial flaking |
| Symmetry | Often high |
| Raw materials | More selective |
| Associated species | *H. ergaster/erectus* |
