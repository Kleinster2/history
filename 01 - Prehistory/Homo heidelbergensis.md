# Homo heidelbergensis

*Homo heidelbergensis* represents the Middle Pleistocene common ancestor of both Neanderthals and modern humans. These "archaic humans" had large brains, sophisticated technology, and may have been the first hominins to routinely hunt large game, control fire, and construct shelters.

## Overview

### Definition

| Attribute | Description |
|-----------|-------------|
| Type specimen | Mauer 1 mandible |
| Type locality | Mauer, Germany |
| Named by | Otto Schoetensack (1908) |
| Time range | ~700-200 ka |
| Brain size | 1100-1400 cc |
| Distribution | Africa, Europe, possibly Asia |

### Alternative Names
- Archaic *Homo sapiens* (older literature)
- *Homo rhodesiensis* (African form)
- *Homo antecessor* (some European fossils)
- "Heidelbergs" (informal)

### Taxonomic Issues

The species is debated:
- Some split African and European forms
- Some synonymize with early *H. sapiens* or *H. erectus*
- Represents a grade rather than a clade?

|---------|--------------|---------------------|--------------|

### Hunting

Evidence for organized large game hunting:

| Site | Age | Evidence |
|------|-----|----------|
| Boxgrove | 500 ka | Rhinoceros butchery |
| Schöningen | 300 ka | Wooden spears + horses |
| Lehringen | 125 ka | Wooden spear in elephant |
| Atapuerca (TD10) | 350 ka | Bison, deer hunting |

### Schöningen Spears (300 ka)

Remarkable wooden spears:
- Eight complete spears preserved
- Up to 2.3 m long
- Carefully shaped
- Weight balanced for throwing
- Found with butchered horses

### Fire Control

## Social Behavior

### Group Living
Evidence suggests:
- Cooperative hunting
- Dependent offspring care
- Knowledge transmission
- Possible food sharing

### Care of Injured

The Sima de los Huesos population:
- Includes injured/disabled individuals
- Survived debilitating injuries
- Implies group support

### Symbolic Behavior?

Limited evidence:
- Bodo skull possibly defleshed
- Some ochre use
- No clear art or ornamentation
- Possible pre-symbolic stage

## Key Sites

## Evolution

### Origins

*H. heidelbergensis* likely evolved from:
- African *H. erectus/ergaster*
- Or from an unknown intermediate
- Possibly via *H. antecessor* in Europe

### Descendants

```
                H. heidelbergensis
                        │
           ┌────────────┼────────────┐
           │            │            │
           ▼            ▼            ▼
      Neanderthals  Denisovans   H. sapiens
      (Europe)       (Asia)      (Africa)
```

### Sima de los Huesos DNA

Ancient DNA analysis revealed:
- Mitochondrial DNA: Denisovan-like
- Nuclear DNA: Closer to Neanderthals
- Complex population history
- Gene flow between lineages

## Regional Variants

### European *H. heidelbergensis*

Features trending toward Neanderthals:
- Mid-facial prognathism
- Occipital morphology
- Large brow ridges
- Large nasal aperture

### African *H. heidelbergensis* (*H. rhodesiensis*)

Features trending toward *H. sapiens*:
- Some facial reduction
- Larger brains
- Less derived occipital

### The Species Problem

| View | Proposal |
|------|----------|
| Lumper | All are *H. heidelbergensis* |
| Splitter | *H. heidelbergensis* (Europe), *H. rhodesiensis* (Africa) |
| Chronospecies | Grade in transition |
| Cladist | Para/polyphyletic grouping |

| *H. heidelbergensis* | 1100-1400 cc |
| Neanderthals | 1200-1700 cc |

### Cognitive Abilities

Evidence suggests:
- Complex multi-step tool making
- Organized hunting of dangerous game
- Fire control and use
- Possible shelter construction
- Teaching and learning
- Long-distance planning
