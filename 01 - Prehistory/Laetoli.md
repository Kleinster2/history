# Laetoli

Laetoli is a paleontological site in northern Tanzania famous for preserving 3.66-million-year-old hominin footprints—the oldest definitive evidence of bipedal walking. The trackways, along with fossil teeth and jaws, demonstrate that *Australopithecus afarensis* walked fully upright more than a million years before the first stone tools.

### The Main Trackway (Site G)

| Feature | Description |
|---------|-------------|
| Length | ~27 m exposed |
| Individuals | 2-3 hominins |
| Trails | G1 (larger), G2/G3 (overlapping smaller) |
| Age | 3.66 Ma |
| Formation | Prints in volcanic ash, rain-cemented |

### Print Characteristics

| Feature | Observation | Significance |
|---------|-------------|--------------|
| Big toe | In line with other toes | Not ape-like grasping |
| Arch | Present | Modern foot structure |
| Heel strike | Clear | Human-like gait |
| Push-off | From toe | Bipedal mechanics |
| Stride length | Consistent | Steady walking |
| Depth | Both feet equal | Upright, not stooped |

### What the Prints Show

| Feature | Interpretation |
|---------|----------------|
| Fully bipedal | Walking upright, not knuckle-walking |
| Modern gait | Heel-strike to toe-off |
| Weight transfer | Human-like |
| Direction | Walking together |
| No hand prints | Arms not used for locomotion |

### Interpretation

| Question | Answer |
|----------|--------|
| Who made them? | *Australopithecus afarensis* |
| How many? | Probably 3 individuals |
| Relationship? | Unknown (family group?) |
| Speed | Slow walk |
| Duration | Brief moment in time |

## Formation of the Footprints

### The Volcanic Eruption

| Stage | Event |
|-------|-------|
| 1 | Sadiman volcano erupts |
| 2 | Fine ash falls on landscape |
| 3 | Light rain dampens ash |
| 4 | Hominins and animals walk across |
| 5 | More ash falls, covering prints |
| 6 | Ash hardens to tuff |
| 7 | Preserved for 3.66 million years |

### Why Preserved

| Factor | Role |
|--------|------|
| Ash chemistry | Calcium-rich, cement-like |
| Moisture | Rain created perfect consistency |
| Timing | Brief window for walking |
| Burial | Rapid covering by more ash |
| Hardening | Carbonatite tuff formation |

## Scientific Significance

### What Laetoli Proved

| Finding | Significance |
|---------|--------------|
| Early bipedalism | Walking upright by 3.66 Ma |
| Modern gait | Not transitional shuffling |
| Before tools | Million+ years before stone tools |
| Before big brains | Small-brained but bipedal |

### Impact on Understanding

| Old View | New Understanding |
|----------|-------------------|
| Bipedalism recent | Fully developed by 3.66 Ma |
| Progressive | Already modern-looking gait |
| Tool-driven | Preceded tool use |
| Brain-first | Bipedalism preceded brain expansion |
