# Olduvai Gorge

Olduvai Gorge is the most famous paleoanthropological site in the worldâ€”a 48-kilometer ravine in Tanzania's Serengeti Plain that has yielded a nearly continuous record of human evolution spanning 2 million years. Excavated primarily by Louis and Mary Leakey, Olduvai has produced iconic fossils and defined major stone tool industries.

## Overview

| Attribute | Detail |
|-----------|--------|
| Location | Northern Tanzania, Ngorongoro Conservation Area |
| Type | Exposed lake-margin sediments |
| Age range | ~2.0 Ma to ~15,000 years |
| Length | ~48 km |
| Depth | Up to 90 m |
| Discovered | 1911 (Wilhelm Kattwinkel) |
| Major excavators | Louis and Mary Leakey |
| UNESCO status | World Heritage Site (Ngorongoro) |

### Etymology

- Maasai name: **Oldupai** (referring to wild sisal plant)
- German/English corruption: "Olduvai"
- Also spelled: Oldoway (historical)

## Geology

### Formation

| Process | Description |
|---------|-------------|
| Lake basin | Ancient lake system |
| Sedimentation | 2+ Ma of deposits |
| Volcanism | Ash layers for dating |
| Rift valley | Faulting and uplift |
| Erosion | River cut the gorge ~500 ka |
| Exposure | Revealed fossil beds |

### Stratigraphy

The gorge exposes seven major geological units (beds):

| Bed | Age (Ma) | Characteristics | Key Fossils |
|-----|----------|-----------------|-------------|
| Bed I | 2.0-1.75 | Lake margin, lava | OH 5, OH 7, Oldowan |
| Bed II | 1.75-1.2 | Lake shrinking | OH 9, Acheulean begins |
| Bed III | 1.2-0.8 | Dry conditions | Limited fossils |
| Bed IV | 0.8-0.6 | Renewed lake | Acheulean |
| Masek Beds | 0.6-0.4 | Ash deposits | |
| Ndutu Beds | 0.4-0.032 | Varied | Modern humans |
| Naisiusiu Beds | 0.022-0.015 | Recent | |

## Major Discoveries

### Hominin Fossils

| Specimen | Bed | Age (Ma) | Species | Significance |
|----------|-----|----------|---------|--------------|
| OH 5 ("Zinj") | I | 1.75 | *Paranthropus boisei* | First major find |
| OH 7 | I | 1.8 | *Homo habilis* | Type specimen |
| OH 9 | II | 1.4 | *Homo erectus* | Massive cranium |
| OH 13 | II | 1.6-1.4 | *Homo habilis* | "Cinderella" |
| OH 24 | I | 1.8 | *Homo habilis* | "Twiggy" |
| OH 62 | I | 1.8 | *Homo habilis* | Partial skeleton |

### OH 5: "Zinjanthropus" / "Nutcracker Man"

| Attribute | Detail |
|-----------|--------|
| Discoverer | Mary Leakey (July 17, 1959) |
| Original name | *Zinjanthropus boisei* |
| Current name | *Paranthropus boisei* |
| Age | 1.75 Ma |
| Features | Massive skull, huge molars |
| Significance | First major hominin at Olduvai |

### OH 7: Type Specimen of *Homo habilis*

| Attribute | Detail |
|-----------|--------|
| Discovery | 1960 (Jonathan Leakey) |
| Elements | Partial skull, hand bones |
| Named | *Homo habilis* ("handy man") |
| Brain size | ~680 cc |
| Significance | Oldest member of genus *Homo* (at time) |

## Stone Tools

### Oldowan Industry

Named after Olduvai Gorge:

| Feature | Description |
|---------|-------------|
| Age | 2.0-1.7 Ma at Olduvai |
| Type site | Bed I, Olduvai |
| Characteristics | Simple cores, flakes |
| Makers | *H. habilis*, early *H. erectus* |

### Developed Oldowan

| Feature | Description |
|---------|-------------|
| Age | ~1.7-1.5 Ma |
| Location | Lower Bed II |
| Features | More refined than classic Oldowan |

### Acheulean Industry

| Feature | Description |
|---------|-------------|
| Age | 1.5 Ma onward at Olduvai |
| First appearance | Bed II |
| Characteristics | Handaxes, bifaces |
| Makers | *Homo erectus* |

### FLK Zinj Site

Famous living floor in Bed I:

| Feature | Description |
|---------|-------------|
| Age | 1.84 Ma |
| Associated with | OH 5 |
| Contents | Dense bone scatter, stone tools |
| Interpretation | Butchery/activity site |
| Debates | Home base vs. natural accumulation |

## Scientific Significance

### What Olduvai Has Contributed

| Contribution | Significance |
|--------------|--------------|
| Multi-species coexistence | *Homo* and *Paranthropus* together |
| Stone tool sequence | Oldowan to Acheulean transition |
| Behavior evidence | Living floors, butchery sites |
| Dating framework | Volcanic ashes, magnetic reversals |
| Long-term record | 2 million years |

### Methodology

Mary Leakey's meticulous approach:
- Detailed mapping
- Careful excavation
- Recording artifact positions
- Refitting stone tools
- Faunal analysis
- Set standards for the field
