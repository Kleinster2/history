# Language Origins

Language is perhaps humanity's most distinctive traitâ€”a system of symbolic communication that enables abstract thought, cultural transmission, and complex social coordination. When and how language evolved remains one of the most debated questions in human origins research.

## Overview

### What Makes Language Unique

| Feature | Human Language | Animal Communication |
|---------|----------------|---------------------|
| Symbolic | Arbitrary signs | Mostly iconic |
| Productive | Infinite combinations | Fixed repertoire |
| Displacement | Past, future, hypothetical | Here and now |
| Cultural transmission | Learned | Largely innate |
| Grammar | Recursive syntax | Limited structure |
| Intentional | Deliberate teaching | Automatic |

### Components of Language

| Component | Description |
|-----------|-------------|
| Phonology | Sound systems |
| Morphology | Word structure |
| Syntax | Grammar, word order |
| Semantics | Meaning |
| Pragmatics | Context, social use |

## Stages of Language Evolution

### Proposed Stages

| Stage | Characteristics | When |
|-------|-----------------|------|
| 1. Primate calls | Emotional, involuntary | Ancestral |
| 2. Intentional signals | Pointing, gestures | 5+ Ma |
| 3. Protolanguage | Words without grammar | 2+ Ma |
| 4. Early syntax | Simple combinations | 500+ ka |
| 5. Full language | Recursive grammar | 100-50 ka |

### Gradualism vs. Saltation

| View | Argument |
|------|----------|
| Gradual evolution | Continuous selection, intermediate forms |
| Sudden emergence | Genetic mutation, syntax appears quickly |
| Current consensus | Gradual with possible acceleration |
